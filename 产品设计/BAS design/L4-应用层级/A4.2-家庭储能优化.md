## 代码示例

### Python实现

```python
import numpy as np
from scipy.optimize import minimize

class HomeEnergyStorageOptimizer:
    def __init__(self, pv_capacity, battery_capacity, load_profile):
        self.pv_capacity = pv_capacity  # kW
        self.battery_capacity = battery_capacity  # kWh
        self.load_profile = load_profile  # kW曲线（24小时）

    def optimize_day_ahead(self, pv_forecast, electricity_price):
        """
        日前优化
        Args:
            pv_forecast: 光伏发电预测（24小时）
            electricity_price: 电价（24小时）
        Returns:
            schedule: {充电功率, 放电功率, 电网购电}
        """
        N = 24  # 24小时
        soc = 0.5  # 初始SOC

        # 决策变量：充放电功率
        def objective(x):
            """目标函数：最小化电费"""
            P_battery = x[:N]  # 电池功率（正=充电，负=放电）
            P_grid = x[N:2*N]  # 电网功率（正=购电，负=卖电）

            # 功率平衡
            P_balance = pv_forecast + P_grid - self.load_profile - P_battery

            # 约束违反惩罚
            penalty = np.sum(np.maximum(0, np.abs(P_balance) - 1e-3)) * 1000

            # 电费
            cost = np.sum(P_grid * electricity_price)

            return cost + penalty

        # 约束
        constraints = [
            # SOC约束
            {'type': 'ineq', 'fun': lambda x: 0.1 - self.calculate_soc(x[:N])[0]},  # SOC >= 10%
            {'type': 'ineq', 'fun': lambda x: self.calculate_soc(x[:N])[-1] - 0.9},  # SOC <= 90%
            # 功率约束
            {'type': 'ineq', 'fun': lambda x: x[:N] + 5},  # P_charge >= -5kW（放电）
            {'type': 'ineq', 'fun': lambda x: 5 - x[:N]},  # P_charge <= 5kW（充电）
        ]

        # 优化
        x0 = np.zeros(2*N)
        result = minimize(objective, x0, constraints=constraints, method='SLSQP')

        # 解析结果
        P_battery = result.x[:N]
        P_grid = result.x[N:2*N]

        return {
            'battery_power': P_battery,
            'grid_power': P_grid,
            'soc_profile': self.calculate_soc(P_battery)
        }

    def calculate_soc(self, P_battery):
        """计算SOC曲线"""
        soc = 0.5
        soc_profile = [soc]
        for P in P_battery:
            soc += P * 1 / self.battery_capacity  # 1小时
            soc = np.clip(soc, 0, 1)
            soc_profile.append(soc)
        return np.array(soc_profile[:-1])
```

### 实时调度

```python
def real_time_dispatch(self, pv_actual, load_actual, soc_current, price_current):
    """
    实时调度
    优先级:
      1. 光伏自用
      2. 电池放电（如果电价高）
      3. 电网购电
    """
    # 净负荷
    net_load = load_actual - pv_actual

    if net_load > 0:  # 缺电
        if price_current > 0.8:  # 高电价
            # 电池放电
            P_battery = min(net_load, 5, soc_current * self.battery_capacity)
            P_grid = net_load - P_battery
        else:  # 低电价
            # 电网购电
            P_battery = 0
            P_grid = net_load
    else:  # 盈电
        if soc_current < 0.9:
            # 充电
            P_battery = max(net_load, -5)
            P_grid = 0
        else:  # 电池已满
            # 卖给电网
            P_battery = 0
            P_grid = net_load

    return P_battery, P_grid
```

## 技术特点

### 优势

1. **光储充协同**: 优化光伏、储能、负载的协同运行
2. **电价优化**: 结合分时电价，降低用电成本
3. **自消纳率提升**: 最大化光伏自用比例
4. **100%通用**: 适用于所有类型的电池系统

### 局限性

1. **预测依赖**: 依赖光伏发电和负载预测精度
2. **参数敏感性**: 需要准确标定电池参数
3. **实时性挑战**: 日前优化可能需要实时调整
4. **政策风险**: 需考虑电价政策和补贴变化
