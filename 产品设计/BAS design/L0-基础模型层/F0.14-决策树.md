# F0.14 决策树（Decision Tree）

## 算法概述

决策树是一种树形结构，通过递归划分特征空间进行预测。在电池管理系统中，决策树主要用于：

- **规则提取**：从数据中提取决策规则
- **故障诊断**：可解释的故障分类
- **特征选择**：识别重要特征

## 核心原理

**划分准则**：
- ID3: 信息增益
- C4.5: 信息增益率
- CART: Gini系数 / MSE

**优势**：
- 可解释性强
- 无需特征缩放
- 自动特征选择

## 代码示例

```python
from sklearn.tree import DecisionTreeClassifier, DecisionTreeRegressor, export_text

def decision_tree_soh(X_train, y_train, feature_names):
    """
    决策树回归估计SOH
    """
    dt = DecisionTreeRegressor(max_depth=5, random_state=42)
    dt.fit(X_train, y_train)

    # 提取决策规则
    rules = export_text(dt, feature_names=feature_names)
    print(rules)

    return dt
```

## 应用场景

- **SOH估计**：可解释的预测
- **故障诊断**：基于规则的诊断
- **特征分析**：识别关键特征
