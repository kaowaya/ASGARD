## 算法原理

### 钠电高自放电率

钠离子电池的自放电率（5-8%/月）显著高于锂电池（2-3%/月），需要特殊补偿模型：

$$
SOC_{corrected} = SOC_{measured} \cdot (1 - k_{sd} \cdot t_{idle} \cdot e^{E_a/RT})
$$

其中：
- $k_{sd}$: 自放电速率常数
- $t_{idle}$: 静置时间
- $E_a$: 活化能
- $R$: 气体常数
- $T$: 温度

## 代码示例

### Python实现

```python
import numpy as np

class SodiumIonSelfDischargeCompensation:
    def __init__(self):
        # 钠电自放电参数（需标定）
        self.k_sd_25 = 0.15  # 25°C时的自放电率（%/天）
        self.E_a = 0.3  # 活化能（eV）
        self.R = 8.314e-5  # 气体常数（eV/K）

    def calculate_self_discharge_rate(self, temperature):
        """
        计算自放电率
        Args:
            temperature: 温度（°C）
        Returns:
            sd_rate: 自放电率（%/天）
        """
        T = temperature + 273.15  # 转换为K

        # Arrhenius方程
        k_sd = self.k_sd_25 * np.exp(self.E_a / self.R * (1/T - 1/298.15))

        return k_sd

    def compensate_soc(self, soc, days_idle, temperature):
        """
        钠电自放电补偿
        Args:
            soc: 测量SOC（0-1）
            days_idle: 静置天数
            temperature: 平均温度（°C）
        Returns:
            soc_compensated: 补偿后的SOC
        """
        # 计算自放电率
        sd_rate = self.calculate_self_discharge_rate(temperature)

        # 补偿SOC
        soc_compensated = soc * (1 - sd_rate * days_idle / 100)

        return max(0, min(soc_compensated, 1))

    def predict_soc_after_idle(self, soc_current, days_idle, temperature):
        """
        预测静置后的SOC
        Args:
            soc_current: 当前SOC
            days_idle: 预计静置天数
            temperature: 预计温度（°C）
        Returns:
            soc_predicted: 预测SOC
        """
        # 计算自放电率
        sd_rate = self.calculate_self_discharge_rate(temperature)

        # 预测SOC
        soc_predicted = soc_current * (1 - sd_rate * days_idle / 100)

        return max(0, min(soc_predicted, 1))

    def estimate_idle_time_from_soc(self, soc_initial, soc_current, temperature):
        """
        从SOC变化估计静置时间
        Args:
            soc_initial: 初始SOC
            soc_current: 当前SOC
            temperature: 温度（°C）
        Returns:
            days_idle: 估计的静置天数
        """
        # 计算自放电率
        sd_rate = self.calculate_self_discharge_rate(temperature)

        # 计算静置时间
        if sd_rate > 0:
            days_idle = (1 - soc_current / soc_initial) * 100 / sd_rate
        else:
            days_idle = 0

        return days_idle

    def optimize_storage_conditions(self, soc_target, storage_duration, max_soc_loss=0.1):
        """
        优化储存条件
        Args:
            soc_target: 目标SOC
            storage_duration: 储存时长（天）
            max_soc_loss: 最大允许SOC损失
        Returns:
            conditions: 优化的储存条件
        """
        # 计算最大允许温度
        # max_soc_loss = sd_rate * storage_duration
        # sd_rate = max_soc_loss / storage_duration

        max_sd_rate = max_soc_loss / storage_duration

        # 从sd_rate反推温度
        # sd_rate = k_sd_25 * exp(E_a/R * (1/T - 1/298.15))
        # 1/T = R/E_a * ln(sd_rate/k_sd_25) + 1/298.15

        T_inv = self.R / self.E_a * np.log(max_sd_rate / self.k_sd_25) + 1/298.15
        T_max = 1 / T_inv - 273.15

        # 推荐储存SOC
        recommended_soc = 0.4  # 钠电推荐40%SOC储存

        return {
            'max_temperature': T_max,
            'recommended_soc': recommended_soc
        }
```

## 技术特点

### 优势

1. **钠电专用**: 专门针对钠离子电池高自放电率
2. **温度补偿**: 考虑温度对自放电的影响
3. **物理模型**: 基于Arrhenius方程
4. **实用性强**: 可直接应用于SOC估计

### 局限性

1. **仅适用于钠电**: 不适用于锂电池等
2. **参数标定**: 需要针对具体电池标定参数
3. **模型简化**: 实际自放电机理更复杂
4. **温度影响**: 温度波动影响精度
