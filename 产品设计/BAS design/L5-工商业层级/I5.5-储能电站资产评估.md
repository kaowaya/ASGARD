## Skill概述

储能电站资产评估通过现金流折现（DCF）模型和风险调整算法，计算储能电站的净现值（NPV）、内部收益率（IRR）和投资回收期，为投资者提供科学的决策依据。

### 核心价值
- **投资决策**: 量化评估项目收益，辅助投资决策
- **风险评估**: 分析电价、政策、技术等风险因素
- **资产估值**: 准确评估储能资产的市场价值
- **融资支持**: 为银行贷款、资产证券化提供依据

### 应用场景
- 储能项目投资评估
- 资产并购估值
- 银行贷款审批
- 资产证券化评估

## 算法原理

### 资产价值评估

```python
class StorageAssetValuation:
    def __init__(self):
        pass

    def calculate_npv(self, station_data, discount_rate=0.08):
        """
        计算净现值
        Args:
            station_data: 电站数据
                - capacity: 容量 (MWh)
                - power: 功率 (MW)
                - initial_investment: 初始投资 (万元)
                - om_cost: 运维成本 (万元/年)
                - revenue_model: 收益模型
                - degradation_rate: 衰减率 (%/年)
                - lifetime: 寿命 (年)
            discount_rate: 折现率
        Returns:
            npv: 净现值
            irr: 内部收益率
        """
        capacity = station_data['capacity']
        initial_investment = station_data['initial_investment']
        om_cost = station_data['om_cost']
        degradation_rate = station_data['degradation_rate']
        lifetime = station_data['lifetime']

        # 计算年收益
        annual_revenue = 0
        for year in range(lifetime):
            # 考虑衰减
            effective_capacity = capacity * (1 - degradation_rate)**year

            # 收益来源
            revenue_arbitrage = self.calculate_arbitrage_revenue(
                effective_capacity, station_data['revenue_model']
            )
            revenue_frequency = self.calculate_frequency_revenue(
                station_data['power'], station_data['revenue_model']
            )
            revenue_capacity = self.calculate_capacity_revenue(
                station_data['power'], station_data['revenue_model']
            )

            annual_revenue_year = revenue_arbitrage + revenue_frequency + revenue_capacity

            # 现金流
            if year == 0:
                cashflow = -initial_investment
            else:
                cashflow = annual_revenue_year - om_cost

            # 折现
            pv = cashflow / (1 + discount_rate)**year
            annual_revenue += pv

        npv = annual_revenue

        # 计算IRR（内部收益率）
        irr = self.calculate_irr(station_data)

        return {
            'npv': npv,
            'irr': irr,
            'payback_period': self.calculate_payback_period(station_data)
        }

    def calculate_arbitrage_revenue(self, capacity, revenue_model):
        """计算套利收益"""
        # 简化：峰谷价差 × 循环次数 × 容量
        price_spread = revenue_model['peak_price'] - revenue_model['valley_price']
        cycles_per_year = 300  # 年循环次数
        revenue = price_spread * cycles_per_year * capacity
        return revenue

    def calculate_frequency_revenue(self, power, revenue_model):
        """计算调频收益"""
        # 简化：调频里程 × 调频价格
        frequency_revenue = power * revenue_model['frequency_price'] * 24 * 365
        return frequency_revenue

    def calculate_capacity_revenue(self, power, revenue_model):
        """计算容量收益"""
        # 简化：容量价格 × 可用容量
        capacity_revenue = power * revenue_model['capacity_price'] * 365
        return capacity_revenue

    def calculate_irr(self, station_data):
        """计算内部收益率"""
        # 使用二分法求解IRR
        low, high = 0, 1
        for _ in range(100):
            mid = (low + high) / 2
            npv = self.calculate_npv(station_data, mid)
            if npv > 0:
                low = mid
            else:
                high = mid
        return mid

    def calculate_payback_period(self, station_data):
        """计算回收期"""
        annual_revenue = self.calculate_arbitrage_revenue(
            station_data['capacity'], station_data['revenue_model']
        )
        payback_years = station_data['initial_investment'] / annual_revenue
        return payback_years
```

### 风险调整

```python
class RiskAdjustment:
    def __init__(self):
        # 风险因子
        self.risk_factors = {
            'price_volatility': 0.3,    # 电价波动风险
            'policy_change': 0.2,       # 政策变化风险
            'technology_degradation': 0.3,  # 技术衰减风险
            'market_competition': 0.2   # 市场竞争风险
        }

    def calculate_risk_adjusted_npv(self, base_npv, risk_scores):
        """
        计算风险调整后的NPV
        Args:
            base_npv: 基础NPV
            risk_scores: 风险评分 {price_volatility: 0.8, ...}
        Returns:
            adjusted_npv: 风险调整后的NPV
        """
        # 计算加权风险评分
        total_risk = sum([
            self.risk_factors[key] * risk_scores[key]
            for key in self.risk_factors
        ])

        # 风险调整
        risk_premium = total_risk * 0.05  # 风险溢价5%
        adjusted_npv = base_npv * (1 - risk_premium)

        return {
            'adjusted_npv': adjusted_npv,
            'risk_premium': risk_premium,
            'risk_level': self.get_risk_level(total_risk)
        }

    def get_risk_level(self, total_risk):
        """获取风险等级"""
        if total_risk < 0.3:
            return '低风险'
        elif total_risk < 0.6:
            return '中等风险'
        else:
            return '高风险'
```

## 技术特点

### 1. 多维度收益模型
- **套利收益**: 峰谷电价套利
- **调频收益**: 参与电网调频服务
- **容量收益**: 提供备用容量
- **补贴收益**: 政府补贴政策

### 2. 动态衰减模型
- **容量衰减**: 每年衰减率2-3%
- **效率衰减**: 循环效率下降
- **寿命预测**: 基于DOD和温度

### 3. 风险评估
- **电价风险**: 电价波动对收益的影响
- **政策风险**: 补贴政策变化
- **技术风险**: 电池性能衰减
- **市场风险**: 竞争加剧

## 商业价值

### 评估报告内容
1. **项目概况**: 容量、功率、投资额
2. **收益分析**: 各收益来源明细
3. **现金流表**: 10年现金流预测
4. **敏感性分析**: 电价、成本变化影响
5. **风险评估**: 风险因子和应对策略
6. **投资建议**: 推荐/不推荐
