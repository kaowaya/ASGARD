# F0.30 Attention注意力机制

## 算法概述

Attention机制允许模型关注输入的重要部分，提供可解释性。在电池管理系统中，Attention可用于：

- **关键时间步识别**：识别影响预测的关键时刻
- **特征重要性**：量化特征贡献
- **可解释性**：可视化模型决策
- **多模态融合**：融合不同模态数据

## Attention公式

```
Attention(Q, K, V) = softmax(Q·K^T / √d_k) · V

其中：
- Q (Query): 查询
- K (Key): 键
- V (Value): 值
- d_k: 维度
```

## 代码示例

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class AttentionLayer(nn.Module):
    """
    通用Attention层
    """
    def __init__(self, hidden_dim):
        super().__init__()

        self.attention = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.Tanh(),
            nn.Linear(hidden_dim, 1)
        )

    def forward(self, lstm_output, mask=None):
        """
        lstm_output: (batch, seq_len, hidden_dim)

        返回:
            context: (batch, hidden_dim)
            weights: (batch, seq_len, 1)
        """
        # 计算注意力权重
        attention_weights = self.attention(lstm_output)  # (batch, seq_len, 1)

        if mask is not None:
            attention_weights = attention_weights.masked_fill(mask == 0, -1e9)

        attention_weights = F.softmax(attention_weights, dim=1)

        # 加权求和
        context = torch.sum(attention_weights * lstm_output, dim=1)  # (batch, hidden_dim)

        return context, attention_weights

class AttentiveModel(nn.Module):
    """
    基于Attention的预测模型
    """
    def __init__(self, input_dim, hidden_dim):
        super().__init__()

        self.lstm = nn.LSTM(input_dim, hidden_dim, batch_first=True)
        self.attention = AttentionLayer(hidden_dim)
        self.fc = nn.Linear(hidden_dim, 1)

    def forward(self, x):
        # LSTM
        lstm_out, _ = self.lstm(x)  # (batch, seq_len, hidden_dim)

        # Attention
        context, attn_weights = self.attention(lstm_out)

        # 预测
        output = self.fc(context)

        return output, attn_weights
```
