## 算法原理

### 1. 高斯过程回归

```
目标: 预测组装后电芯的一致性指标

核函数（RBF）:
  k(x_i, x_j) = σ_f² · exp(-||x_i - x_j||² / 2l²)

预测:
  f* | X, y, x* ~ N(μ*, Σ*)

  均值: μ* = k*ᵀ(K + σ_n²I)⁻¹y
  方差: Σ* = k(x*, x*) - k*ᵀ(K + σ_n²I)⁻¹k*

其中:
  K: 训练数据的核矩阵
  k*: 测试点与训练数据的核向量
  σ_n²: 噪声方差
```

### 2. 贝叶斯优化

```
目标: 优化组装工艺参数

采集函数（EI）:
  EI(x) = E[max(f(x) - f(x_best), 0)]

  其中:
    f(x): GPR预测的性能
    f(x_best): 当前最佳性能

优化流程:
  1. 初始化: 随机采样10组参数
  2. 训练GPR模型
  3. 最大化EI，推荐下一组参数
  4. 实验验证
  5. 更新GPR模型
  6. 重复2-5，直到收敛
```

## 代码示例

```python
from sklearn.gaussian_process import GaussianProcessRegressor
from sklearn.gaussian_process.kernels import RBF, ConstantKernel
import numpy as np

# 定义核函数
kernel = ConstantKernel(1.0) * RBF(length_scale=1.0)

# GPR模型
gpr = GaussianProcessRegressor(
    kernel=kernel,
    alpha=1e-6,
    normalize_y=True,
    n_restarts_optimizer=10
)

# 训练数据（组装参数 → 一致性指标）
X_train = np.array([
    [压力1, 温度1, 时间1],
    [压力2, 温度2, 时间2],
    ...
])
y_train = np.array([一致性1, 一致性2, ...])

# 训练
gpr.fit(X_train, y_train)

# 预测
X_test = np.array([[新压力, 新温度, 新时间]])
y_pred, sigma = gpr.predict(X_test, return_std=True)
print(f"预测一致性: {y_pred[0]:.3f} ± {sigma[0]:.3f}")
```
