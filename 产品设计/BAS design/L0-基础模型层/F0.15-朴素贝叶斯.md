# F0.15 朴素贝叶斯（Naive Bayes）

## 算法概述

朴素贝叶斯基于贝叶斯定理和特征独立性假设，是一种简单但有效的概率分类算法。在电池管理系统中，可用于：

- **快速分类**：电池状态分类
- **文本分类**：故障报告分类
- **实时诊断**：在线故障诊断

## 核心原理

**贝叶斯定理**：
```
P(y|X) = P(X|y) · P(y) / P(X)
```

**朴素假设**：特征相互独立

**变体**：
- 高斯朴素贝叶斯：连续特征
- 多项式朴素贝叶斯：离散特征
- 伯努利朴素贝叶斯：二值特征

## 代码示例

```python
from sklearn.naive_bayes import GaussianNB

def battery_state_classification_nb(X_train, y_train, X_test):
    """
    朴素贝叶斯分类电池状态
    """
    nb = GaussianNB()
    nb.fit(X_train, y_train)

    # 预测
    labels_pred = nb.predict(X_test)

    # 概率
    probs = nb.predict_proba(X_test)

    return labels_pred, probs
```

## 应用场景

- **状态分类**：充电/放电/静置
- **故障分类**：正常/异常
- **快速诊断**：实时分类
