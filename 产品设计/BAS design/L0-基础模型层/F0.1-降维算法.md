# F0.1 降维算法（PCA、t-SNE、UMAP）

## 算法概述

降维算法是将高维数据映射到低维空间的技术，旨在保留数据的关键特征结构。在电池管理系统中，降维算法主要用于：

- **特征可视化**：将高维电池数据映射到2D/3D空间进行可视化分析
- **数据压缩**：减少存储空间和计算复杂度
- **聚类预处理**：提高聚类算法的效率和效果
- **噪声去除**：过滤次要特征，保留主要信息

## 算法对比

| 算法 | 速度 | 全局结构保持 | 局部结构保持 | 适用场景 |
|------|------|-------------|-------------|---------|
| **PCA** | 快 | 优秀 | 一般 | 线性降维、主成分分析 |
| **t-SNE** | 慢 | 差 | 优秀 | 高维可视化 |
| **UMAP** | 中 | 良好 | 优秀 | 大规模数据降维 |

## 核心原理

### PCA（主成分分析）

**核心思想**：找到数据方差最大的方向

**步骤**：
1. 数据标准化：`X_std = (X - μ) / σ`
2. 计算协方差矩阵：`C = X_std^T · X_std / (n-1)`
3. 特征值分解：`C · v_i = λ_i · v_i`
4. 选择前k个主成分：`V_k = [v_1, v_2, ..., v_k]`
5. 降维：`X_low = X_std · V_k`

**数学公式**：
```
X_low = X · W_k
其中 W_k 是前k个特征向量组成的矩阵
```

### t-SNE（t-分布随机邻域嵌入）

**核心思想**：保持高维空间和低维空间的概率分布相似

**步骤**：
1. 计算高维空间相似度：
   ```
   p_{j|i} = exp(-||x_i - x_j||² / 2σ_i²) / Σ_{k≠i} exp(-||x_i - x_k||² / 2σ_i²)
   p_{ij} = (p_{j|i} + p_{i|j}) / 2n
   ```

2. 计算低维空间相似度（t分布）：
   ```
   q_{ij} = (1 + ||y_i - y_j||²)^{-1} / Σ_{k≠l} (1 + ||y_k - y_l||²)^{-1}
   ```

3. 最小化KL散度：
   ```
   Loss = KL(P||Q) = Σ_{i≠j} p_{ij} log(p_{ij} / q_{ij})
   ```

### UMAP（均匀流形近似与投影）

**核心思想**：假设数据均匀分布在黎曼流形上

**步骤**：
1. 构建加权k近邻图（高维空间）
2. 在低维空间构建类似的图结构
3. 优化两个图的相似性

**优势**：
- 比t-SNE快得多
- 更好地保持全局结构
- 可处理大规模数据集

## 代码示例

```python
from sklearn.decomposition import PCA
from sklearn.manifold import TSNE
import umap
import numpy as np

def dimensionality_reduction(X, method='PCA', n_components=2):
    """
    降维算法统一接口

    参数:
        X: 高维特征矩阵 (n_samples, n_features)
        method: 'PCA', 'TSNE', or 'UMAP'
        n_components: 目标维度

    返回:
        X_low: 低维嵌入 (n_samples, n_components)
    """

    if method == 'PCA':
        # PCA: 适合线性可分数据，速度快
        model = PCA(n_components=n_components)
        X_low = model.fit_transform(X)

        # 解释方差比
        explained_variance = model.explained_variance_ratio_
        print(f"前{n_components}个主成分解释方差: {explained_variance.sum():.2%}")

    elif method == 'TSNE':
        # t-SNE: 适合可视化，但速度慢
        model = TSNE(
            n_components=n_components,
            perplexity=30,  # 建议值：5-50
            n_iter=1000,
            random_state=42
        )
        X_low = model.fit_transform(X)

    elif method == 'UMAP':
        # UMAP: 平衡速度和质量
        model = umap.UMAP(
            n_components=n_components,
            n_neighbors=15,  # 控制局部/全局结构平衡
            min_dist=0.1,    # 控制点的紧凑程度
            metric='euclidean'
        )
        X_low = model.fit_transform(X)

    return X_low

# 应用场景示例
def battery_cell_clustering_preprocessing(voltage_curves, current_curves, temp_curves):
    """
    电池电芯聚类的预处理

    步骤：
    1. 提取时序特征
    2. PCA降维
    3. 聚类
    """
    # 1. 特征提取
    features = []
    for v, i, t in zip(voltage_curves, current_curves, temp_curves):
        # 统计特征
        feat = [
            np.mean(v), np.std(v), np.max(v), np.min(v),  # 电压特征
            np.mean(i), np.std(i),                         # 电流特征
            np.mean(t), np.max(t) - np.min(t),             # 温度特征
            np.corrcoef(v, i)[0,1]                         # 电压-电流相关性
        ]
        features.append(feat)

    X = np.array(features)

    # 2. PCA降维（从9维降到3维）
    X_pca = dimensionality_reduction(X, method='PCA', n_components=3)

    # 3. 可视化（可选）
    X_tsne = dimensionality_reduction(X, method='TSNE', n_components=2)

    return X_pca, X_tsne
```

## 应用场景

### 场景1：电芯分级可视化
```
输入：1000个电芯的充电曲线特征
处理：t-SNE降维到2D
输出：散点图，不同等级电芯自动聚类
价值：快速识别异常电芯
```

### 场景2：储能电站数据压缩
```
输入：10000个电芯的实时电压、电流、温度（100维）
处理：PCA降维到10维
输出：压缩后的特征向量
价值：节省90%存储空间，加速后续分析
```

## 部署建议

### 云端部署
- **适用场景**：大规模数据处理、离线分析
- **优势**：计算资源充足，可处理超大规模数据集
- **推荐配置**：16GB+ RAM，多核CPU

### 边缘端部署
- **适用场景**：实时数据预处理、本地特征提取
- **推荐算法**：PCA（计算快，资源需求低）
- **优化方案**：
  - 使用增量PCA（IncrementalPCA）处理流式数据
  - 量化降维后的特征向量
  - 预训练PCA模型，仅做推理

## 性能优化

1. **数据预处理**：
   - 标准化：`X_std = (X - mean) / std`
   - 去除异常值：3σ原则

2. **参数调优**：
   - PCA：选择解释方差达到95%的主成分数量
   - t-SNE：perplexity参数建议5-50
   - UMAP：n_neighbors控制局部/全局平衡

3. **加速技巧**：
   - PCA：使用随机SVD（randomized SVD）
   - t-SNE：先用PCA降维到50维，再运行t-SNE
   - UMAP：增加n_jobs参数并行计算
