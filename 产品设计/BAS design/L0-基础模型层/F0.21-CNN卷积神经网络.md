# F0.21 CNN卷积神经网络

## 算法概述

卷积神经网络（CNN）是一种专门处理网格结构数据的深度学习架构。在电池管理系统中，CNN主要用于：

- **极片缺陷检测**：生产线质量检测
- **电压模式识别**：1D时序分类
- **故障图像识别**：热失控、析锂识别

## 典型架构

```
Input → Conv1 → ReLU → Pool1 → Conv2 → ReLU → Pool2
      → FC1 → FC2 → Output
```

## 核心原理

### 卷积层

**输出特征图**：
```
Y[i,j] = Σ_{m,n} X[i+m, j+n] · K[m,n] + b

其中：
- X: 输入特征图
- K: 卷积核（滤波器）
- b: 偏置
- Y: 输出特征图
```

**优势**：
- 局部连接（减少参数）
- 权值共享（平移不变性）
- 层次特征提取（边缘→纹理→对象）

### 池化层

**最大池化**：
```
Y[i,j] = max_{m,n∈R_{i,j}} X[m,n]
```

**平均池化**：
```
Y[i,j] = mean_{m,n∈R_{i,j}} X[m,n]
```

**作用**：
- 降维（减少计算量）
- 平移不变性
- 防止过拟合

## 代码示例

```python
import torch
import torch.nn as nn

class BatteryCNN(nn.Module):
    """
    电池电压模式识别CNN

    输入：电压时序 (batch, 1, sequence_length)
    输出：分类概率 (batch, n_classes)
    """
    def __init__(self, input_dim, n_classes):
        super().__init__()

        # 1D CNN（处理时序数据）
        self.conv1 = nn.Conv1d(
            in_channels=1,
            out_channels=16,
            kernel_size=5,
            stride=1,
            padding=2
        )
        self.relu = nn.ReLU()
        self.pool1 = nn.MaxPool1d(kernel_size=2)

        self.conv2 = nn.Conv1d(16, 32, kernel_size=5, padding=2)
        self.pool2 = nn.MaxPool1d(kernel_size=2)

        # 全连接层
        self.fc1 = nn.Linear(32 * (input_dim // 4), 64)
        self.fc2 = nn.Linear(64, n_classes)

    def forward(self, x):
        # Conv1
        x = self.conv1(x)
        x = self.relu(x)
        x = self.pool1(x)

        # Conv2
        x = self.conv2(x)
        x = self.relu(x)
        x = self.pool2(x)

        # Flatten
        x = x.view(x.size(0), -1)

        # FC
        x = self.fc1(x)
        x = self.relu(x)
        x = self.fc2(x)

        return x

# 应用：极片缺陷检测
class DefectDetectionCNN(nn.Module):
    """
    2D CNN：极片缺陷检测

    输入：极片图像 (batch, 3, H, W)
    输出：缺陷类别 + 位置
    """
    def __init__(self, n_classes):
        super().__init__()

        # Backbone（类似VGG）
        self.features = nn.Sequential(
            nn.Conv2d(3, 64, 3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),

            nn.Conv2d(64, 128, 3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),

            nn.Conv2d(128, 256, 3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),
        )

        # 分类头
        self.classifier = nn.Sequential(
            nn.Linear(256 * (H//8) * (W//8), 512),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(512, n_classes)
        )

    def forward(self, x):
        x = self.features(x)
        x = x.view(x.size(0), -1)
        x = self.classifier(x)
        return x
```

## 应用场景

### 场景1：极片缺陷AI检测
```
输入：极片涂布图像（高分辨率线扫描）
处理：YOLO/Mask R-CNN（目标检测）
输出：缺陷类别 + 位置 + 严重程度评分
价值：实时检测，漏检率 <0.1%
```

### 场景2：电压模式识别
```
输入：充放电电压曲线（1D时序）
处理：1D-CNN
输出：正常/析锂/内短路/异常
价值：快速故障分类
```

## 部署建议

### 云端部署
- **训练**：大规模数据训练
- **推理**：高精度检测

### 边缘端部署
- **量化**：INT8量化降低计算量
- **剪枝**：去除冗余连接
- **轻量化模型**：MobileNet、ShuffleNet

## 性能优化

1. **数据增强**：
   - 图像：旋转、翻转、裁剪
   - 时序：时间扭曲、幅值缩放

2. **正则化**：
   - Dropout：防止过拟合
   - BatchNorm：加速收敛

3. **优化器**：
   - Adam：自适应学习率
   - SGD+Momentum：更稳定
