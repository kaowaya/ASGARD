# F0.41 粒子群优化（PSO）

## 算法概述

PSO模拟鸟群觅食行为，通过个体和社会学习寻找最优解。在电池管理系统中，PSO主要用于：

- **参数辨识**：电池模型参数
- **模型训练**：神经网络权重
- **多目标优化**：充电策略

## PSO更新公式

```
v_i(t+1) = w·v_i(t) + c1·r1·(pbest_i - x_i(t)) + c2·r2·(gbest - x_i(t))
x_i(t+1) = x_i(t) + v_i(t+1)

参数说明：
- w: 惯性权重（0.8）
- c1: 个体学习因子（2.0）
- c2: 社会学习因子（2.0）
- r1, r2: 随机数（0-1）
- pbest_i: 粒子i的历史最优位置
- gbest: 全局最优位置
```

## 代码示例

```python
import numpy as np

def pso(objective_func, bounds, n_particles=30, max_iter=100):
    """
    粒子群优化

    参数：
        objective_func: 目标函数（最小化）
        bounds: 参数边界 [(min_1, max_1), ..., (min_n, max_n)]
        n_particles: 粒子数量
        max_iter: 最大迭代次数
    """
    n_vars = len(bounds)

    # 1. 初始化粒子群
    particles = np.random.uniform(
        low=[b[0] for b in bounds],
        high=[b[1] for b in bounds],
        size=(n_particles, n_vars)
    )
    velocities = np.zeros((n_particles, n_vars))

    # 初始化个体最优和全局最优
    pbest = particles.copy()
    pbest_fitness = np.array([objective_func(p) for p in particles])
    gbest_idx = np.argmin(pbest_fitness)
    gbest = particles[gbest_idx].copy()

    # PSO参数
    w = 0.8  # 惯性权重
    c1 = 2.0  # 个体学习因子
    c2 = 2.0  # 社会学习因子

    # 2. 迭代优化
    for iter in range(max_iter):
        for i in range(n_particles):
            # 更新速度
            r1, r2 = np.random.rand(2)
            velocities[i] = (w * velocities[i] +
                            c1 * r1 * (pbest[i] - particles[i]) +
                            c2 * r2 * (gbest - particles[i]))

            # 更新位置
            particles[i] += velocities[i]

            # 边界处理
            particles[i] = np.clip(
                particles[i],
                [b[0] for b in bounds],
                [b[1] for b in bounds]
            )

            # 评估适应度
            fitness = objective_func(particles[i])

            # 更新个体最优
            if fitness < pbest_fitness[i]:
                pbest[i] = particles[i].copy()
                pbest_fitness[i] = fitness

            # 更新全局最优
            if fitness < pbest_fitness[gbest_idx]:
                gbest_idx = i
                gbest = particles[i].copy()

        # 打印进度
        if iter % 10 == 0:
            print(f"Iteration {iter}, Best Fitness: {pbest_fitness[gbest_idx]:.6f}")

    return gbest, pbest_fitness[gbest_idx]

# 应用：电池模型参数辨识

# 参数边界

print(f"最小误差: {best_error}")
```
