# F0.18 极端随机树（ExtraTrees）

## 算法概述

ExtraTrees（Extremely Randomized Trees）是随机森林的变体，特征划分更加随机。在电池管理系统中，可用于：

- **SOH估计**：快速回归预测
- **特征重要性**：特征选择
- **降维**：特征提取

## 核心原理

**vs 随机森林**：
- 随机森林：随机特征 + 最佳分割点
- ExtraTrees：随机特征 + 随机分割点

**优势**：
- 偏差略高，方差更低
- 训练速度更快
- 更好的泛化能力

## 代码示例

```python
from sklearn.ensemble import ExtraTreesRegressor

def soh_extratrees(X_train, y_train, X_test):
    """
    ExtraTrees估计SOH
    """
    et = ExtraTreesRegressor(
        n_estimators=100,
        max_depth=10,
        min_samples_split=5,
        random_state=42,
        n_jobs=-1
    )
    et.fit(X_train, y_train)
    soh_pred = et.predict(X_test)
    importance = et.feature_importances_
    return soh_pred, importance
```

## 应用场景

- **SOH估计**：快速预测
- **特征分析**：特征重要性
- **大规模数据**：快速训练
