## Skill概述

工厂配储优化通过分析工厂负荷特性，优化储能容量配置和运行策略，实现峰谷电价套利、需量电费管理和可再生能源消纳，显著降低工厂用电成本。

### 核心价值
- **降低电费**: 节省电费15-25%
- **削峰填谷**: 削减峰值负荷，降低需量电费
- **容量优化**: 科学配置储能容量，最大化投资回报
- **绿电消纳**: 提升光伏自用率，减少外购电

### 应用场景
- 制造业工厂
- 数据中心
- 大型商业综合体
- 高耗能企业（钢铁、化工、水泥等）

## 算法原理

### 工厂负荷特性分析

```python
class FactoryLoadAnalyzer:
    def __init__(self):
        pass

    def analyze_load_profile(self, power_data, datetime_index):
        """
        分析工厂负荷特性
        Returns:
            characteristics: {
                'peak_load': 峰值负荷
                'base_load': 基础负荷
                'load_factor': 负荷率
                'peak_shaving_potential': 削峰潜力
            }
        """
        # 统计分析
        peak_load = np.max(power_data)
        base_load = np.min(power_data)
        avg_load = np.mean(power_data)
        load_factor = avg_load / peak_load

        # 峰值时段识别
        peak_hours = self.identify_peak_hours(power_data, datetime_index)

        # 削峰潜力
        peak_shaving_potential = np.sum(power_data[peak_hours] - base_load)

        characteristics = {
            'peak_load': peak_load,
            'base_load': base_load,
            'avg_load': avg_load,
            'load_factor': load_factor,
            'peak_hours': peak_hours,
            'peak_shaving_potential': peak_shaving_potential
        }

        return characteristics

    def identify_peak_hours(self, power_data, datetime_index):
        """识别峰值时段"""
        threshold = np.percentile(power_data, 90)  # 前10%
        peak_hours = power_data > threshold
        return peak_hours
```

### 储能优化配置

```python
from scipy.optimize import linprog

class FactoryStorageOptimizer:
    def __init__(self, factory_profile):
        self.factory_profile = factory_profile

    def optimize_storage_capacity(self, load_profile, price_profile, investment_cost):
        """
        优化储能容量配置
        Args:
            load_profile: 负荷曲线（24小时）
            price_profile: 电价曲线（24小时）
            investment_cost: 投资成本（元/kWh）
        Returns:
            optimal_capacity: 最优容量
            optimal_power: 最优功率
            npv: 净现值
        """
        # 遍历不同容量配置
        capacities = np.arange(100, 2000, 100)  # 100-2000kWh
        npvs = []

        for capacity in capacities:
            # 优化给定容量下的运行策略
            revenue = self.optimize_operation(capacity, load_profile, price_profile)

            # 计算NPV（假设10年寿命，折现率8%）
            annual_revenue = revenue * 365  # 年收益
            investment = capacity * investment_cost
            npv = -investment + sum([annual_revenue / (1.08)**i for i in range(1, 11)])
            npvs.append(npv)

        # 选择NPV最大的容量
        optimal_idx = np.argmax(npvs)
        optimal_capacity = capacities[optimal_idx]
        optimal_power = optimal_capacity / 2  # 假设2小时充放电

        return optimal_capacity, optimal_power, npvs[optimal_idx]

    def optimize_operation(self, capacity, load_profile, price_profile):
        """
        优化给定容量下的运行策略
        """
        N = 24

        # 决策变量: [充电功率(kW) × 24小时]
        c = load_profile  # 负荷（目标最小化）
        A_eq = np.eye(N)  # 单位矩阵（功率约束）
        b_eq = np.ones(N) * capacity / 2  # 功率上限（2小时充放电）

        # 线性规划求解
        result = linprog(c, A_ub=A_eq, b_ub=b_eq, bounds=(0, capacity/2), method='highs')

        if result.success:
            # 计算收益（套利）
            charge_power = result.x
            discharge_power = -result.x
            soc = np.cumsum(charge_power - discharge_power) / capacity

            # 收益 = 低谷充电成本 + 高峰放电收益
            revenue = np.sum(charge_power * price_profile * -1) + \
                     np.sum(discharge_power * price_profile)

            return revenue
        else:
            return 0
```

## 技术特点

### 1. 负荷特性分析
- **峰谷识别**: 自动识别工厂的峰值和谷值时段
- **负荷率计算**: 评估工厂用电的平稳性
- **削峰潜力**: 量化储能可削减的峰值负荷

### 2. 容量优化
- **NPV最大化**: 基于净现值选择最优容量
- **投资回报分析**: 计算投资回收期和内部收益率
- **敏感性分析**: 评估电价、负荷变化对收益的影响

### 3. 运行策略优化
- **峰谷套利**: 低谷充电、高峰放电
- **需量管理**: 削减峰值，降低基本电费
- **光伏消纳**: 优先使用光伏发电，提升自用率
