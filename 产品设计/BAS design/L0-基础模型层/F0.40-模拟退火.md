# F0.40 模拟退火（Simulated Annealing）

## 算法概述

模拟退火模拟金属退火过程，通过概率接受劣解跳出局部最优。在电池管理系统中，SA主要用于：

- **参数辨识**：电池模型参数
- **组合优化**：充电策略
- **全局优化**：复杂目标函数

## 算法步骤

```
1. 初始化：随机初始解x，初始温度T
2. 迭代：
    a. 生成邻域解x'（随机扰动）
    b. 计算目标函数差值ΔE = f(x') - f(x)
    c. 接受准则：
        - 如果ΔE < 0，接受x'
        - 否则以概率exp(-ΔE/T)接受x'
    d. 降温：T = α·T（α为降温系数）
3. 终止：温度低于阈值或达到最大迭代次数
```

**优势**：能跳出局部最优（以一定概率接受劣解）

## 代码示例

```python
import numpy as np

def simulated_annealing(objective_func, bounds, n_iterations=1000, temp=100, cooling_rate=0.95):
    """
    模拟退火优化

    参数：
        objective_func: 目标函数（最小化）
        bounds: 参数边界 [(min_1, max_1), ..., (min_n, max_n)]
        n_iterations: 最大迭代次数
        temp: 初始温度
        cooling_rate: 降温系数

    返回：
        best_solution: 最优解
        best_cost: 最优目标函数值
    """
    n_vars = len(bounds)

    # 初始化
    current_solution = np.random.uniform(
        low=[b[0] for b in bounds],
        high=[b[1] for b in bounds]
    )
    current_cost = objective_func(current_solution)

    best_solution = current_solution.copy()
    best_cost = current_cost

    # 退火循环
    for i in range(n_iterations):
        # 生成邻域解
        new_solution = current_solution + np.random.normal(0, 0.1, n_vars)

        # 边界处理
        new_solution = np.clip(
            new_solution,
            [b[0] for b in bounds],
            [b[1] for b in bounds]
        )

        # 计算新目标函数值
        new_cost = objective_func(new_solution)

        # 计算差值
        delta = new_cost - current_cost

        # 接受准则
        if delta < 0 or np.random.rand() < np.exp(-delta / temp):
            current_solution = new_solution
            current_cost = new_cost

            # 更新最优解
            if current_cost < best_cost:
                best_solution = current_solution.copy()
                best_cost = current_cost

        # 降温
        temp *= cooling_rate

        # 打印进度
        if i % 100 == 0:
            print(f"Iteration {i}, Temp: {temp:.2f}, Best Cost: {best_cost:.6f}")

    return best_solution, best_cost
```
