# F0.20 逻辑回归（Logistic Regression）

## 算法概述

逻辑回归是一种线性分类模型，尽管名字叫"回归"，实际上是分类算法。在电池管理系统中，逻辑回归主要用于：

- **二分类**：正常/异常
- **多分类**：多类故障分类
- **概率预测**：故障概率估计

## 核心原理

**Sigmoid函数**：
```
σ(z) = 1 / (1 + e^(-z))
```

**假设函数**：
```
hθ(x) = σ(θ^T · x)
```

**损失函数**（交叉熵）：
```
J(θ) = -1/m Σ[y·log(h) + (1-y)·log(1-h)]
```

**优势**：
- 简单快速
- 可解释性强
- 输出概率
- 适合线性可分数据

## 代码示例

```python
from sklearn.linear_model import LogisticRegression
from sklearn.preprocessing import StandardScaler

def battery_fault_classification_lr(X_train, y_train, X_test):
    """
    逻辑回归故障分类
    """
    # 标准化
    scaler = StandardScaler()
    X_train_std = scaler.fit_transform(X_train)
    X_test_std = scaler.transform(X_test)

    # 训练
    lr = LogisticRegression(
        penalty='l2',        # L2正则化
        C=1.0,              # 正则化强度
        multi_class='ovr',  # 多分类策略
        max_iter=100,
        random_state=42
    )
    lr.fit(X_train_std, y_train)

    # 预测
    labels_pred = lr.predict(X_test_std)

    # 概率
    probs = lr.predict_proba(X_test_std)

    return labels_pred, probs, lr
```

## 应用场景

- **故障分类**：正常/异常
- **风险预测**：故障概率
- **状态分类**：多状态分类
