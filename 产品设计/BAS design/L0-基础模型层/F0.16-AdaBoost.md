# F0.16 AdaBoost（Adaptive Boosting）

## 算法概述

AdaBoost是一种迭代算法，通过调整样本权重来训练一系列弱分类器，最终组合成强分类器。在电池管理系统中，可用于：

- **故障分类**：多类别故障识别
- **状态分类**：电池工作状态分类
- **异常检测**：二分类异常检测

## 核心原理

**核心思想**：关注被错误分类的样本

**步骤**：
1. 初始化样本权重
2. 训练弱分类器
3. 计算分类误差
4. 更新样本权重（增加错误样本权重）
5. 重复2-4
6. 加权投票

## 代码示例

```python
from sklearn.ensemble import AdaBoostClassifier
from sklearn.tree import DecisionTreeClassifier

def fault_classification_adaboost(X_train, y_train, X_test):
    """
    AdaBoost故障分类
    """
    base = DecisionTreeClassifier(max_depth=1)
    ada = AdaBoostClassifier(
        base_estimator=base,
        n_estimators=100,
        learning_rate=1.0,
        random_state=42
    )
    ada.fit(X_train, y_train)
    labels_pred = ada.predict(X_test)
    return labels_pred, ada
```

## 应用场景

- **故障分类**：多类型故障
- **状态分类**：工作状态
- **质量控制**：良品/次品分类
