## 📋 文档概述

本文档描述ASGARD项目的产品能力Demo前端界面设计，旨在向甲方展示ASGARD的核心价值和技术实力，促成商业合作。

### 设计目标

- **主要受众**: 甲方决策者（非技术）+ 技术工程师（混合用户）
- **核心场景**: 产品能力演示（展示ASGARD的智能化能力）
- **技术栈**: 纯Web（HTML+JS+ECharts）+ Tailwind CSS（CDN）
- **关键差异点**: Agent智能编排 + 透明化决策过程

## 🎨 设计系统

### 核心设计哲学

```
采用Apple设计语言的核心理念：
- 极致简约 - 去除一切不必要的元素
- 功能至上 - 每个元素都有明确目的
- 用户友好 - 直觉式操作体验
- 精确美学 - 像素级完美对齐
- 情感共鸣 - 微妙但有意义的交互反馈
```

### 视觉系统

#### 色彩规范

```css
/* 60-30-10 比例 */
背景（60%）: 纯白 #FFFFFF
表面（30%）: 中性灰 #F5F5F5
强调（10%）: 炭黑 #09090B

/* 语义色 */
成功: #22C55E
错误: #EF4444
警告: #F97316
信息: #3B82F6
```

#### 字体系统

```
Display:   clamp(2.25rem, 5vw, 5rem)   /* 36px - 80px */
Headline:  clamp(1.75rem, 4vw, 3rem)   /* 28px - 48px */
Title:     clamp(1.125rem, 2.5vw, 1.5rem) /* 18px - 24px */
Body:      clamp(0.875rem, 2vw, 1.125rem) /* 14px - 18px */
Caption:   clamp(0.75rem, 1.5vw, 0.875rem) /* 12px - 14px */

字体栈: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
```

#### 间距系统

```
基础单位: 8px网格
间距令牌: 4, 8, 16, 24, 32px

视觉分组:
- 相关项目: 4-8px
- 不同组: 16-24px
- 模块间: 24-32px
```

#### 布局系统

```
Bento Grid（便当盒美学）:
- 圆角矩形卡片
- 统一间距（24-32px）
- 最大宽度1120px居中
- 响应式列数（桌面4列，平板2列，移动1列）

响应式断点:
- 移动: <768px
- 平板: 768-1024px
- 桌面: >1024px
```

### 交互系统

```css
/* 微交互 */
卡片hover: translate(-0.5px) + shadow加深
过渡时长: 0.3s ease
触摸区域: ≥48×48px

/* 无障碍 */
支持 prefers-reduced-motion
文本对比度: WCAG AA (≥4.5:1)
焦点指示: 清晰的ring状态
```

   - 结果摘要

## 📱 响应式设计

### 断点规范

```
移动（<768px）:
- 单列堆叠布局
- 汉堡菜单
- 触摸优化（≥48×48px）
- 字号 clamp 自适应

平板（768-1024px）:
- 2列网格
- 底部导航或精简顶部导航
- 适中的触摸目标

桌面（>1024px）:
- 4列网格（首页）
- 完整顶部导航
- 鼠标交互优化
```

### Agent编排页面响应式

```
桌面（>1024px）:
- 左栏 45%, 右栏 55%

平板（768-1024px）:
- 左栏 50%, 右栏 50%

移动（<768px）:
- 分页显示（Tab切换）
- Tab 1: Agent对话
- Tab 2: Workflow DAG
```
