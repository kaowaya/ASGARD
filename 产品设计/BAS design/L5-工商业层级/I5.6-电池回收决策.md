## Skill概述

电池回收决策通过多准则决策分析（MCDA）算法，综合考虑电池健康状态（SOH）、梯次利用价值、材料回收价值和环保价值，为退役电池提供最优的处理方案（继续使用、梯次利用或回收）。

### 核心价值
- **价值最大化**: 选择收益最高的处理方案
- **环保合规**: 符合电池回收法规要求
- **资源循环**: 促进电池材料循环利用
- **风险控制**: 避免不当使用导致的安全风险

### 应用场景
- 储能电站退役电池评估
- 电动汽车电池梯次利用
- 电池资产管理
- 回收企业决策支持

## 算法原理

### 回收决策树

```python
class BatteryRecyclingDecision:
    def __init__(self):
        # 决策权重
        self.weights = {
            'soh': 0.4,        # 健康状态
            'resale_value': 0.3,  # 梯次利用价值
            'recycling_value': 0.2,  # 材料回收价值
            'environmental': 0.1   # 环保价值
        }

    def make_decision(self, battery_data):
        """
        回收决策
        Args:
            battery_data: {
                'soh': 健康状态
                'capacity': 容量
                'age': 使用年限
                'chemistry': 电池类型
            }
        Returns:
            decision: {
                'action': '继续使用' / '梯次利用' / '回收'
                'confidence': 置信度
                'reason': 决策原因
            }
        """
        soh = battery_data['soh']
        age = battery_data['age']

        # 决策树
        if soh > 0.8:
            # 继续使用
            decision = {
                'action': '继续使用',
                'confidence': 0.95,
                'reason': f'电池健康状态良好(SOH={soh:.1%})，可继续使用'
            }
        elif 0.6 <= soh <= 0.8:
            # 梯次利用评估
            cascade_value = self.evaluate_cascade_value(battery_data)
            if cascade_value > 0.6:
                decision = {
                    'action': '梯次利用',
                    'confidence': 0.85,
                    'reason': f'适合梯次利用(储能系统)，价值评分{cascade_value:.2f}'
                }
            else:
                decision = {
                    'action': '回收',
                    'confidence': 0.7,
                    'reason': '梯次利用价值较低，建议直接回收'
                }
        else:  # soh < 0.6
            # 回收
            recycling_value = self.estimate_recycling_value(battery_data)
            decision = {
                'action': '回收',
                'confidence': 0.9,
                'reason': f'电池接近寿命终点，建议回收(预期价值¥{recycling_value:.0f})'
            }

        return decision

    def evaluate_cascade_value(self, battery_data):
        """
        评估梯次利用价值
        考虑因素:
          1. 剩余容量
          2. 内阻
          3. 循环寿命
          4. 市场需求
        """
        soh = battery_data['soh']
        capacity = battery_data['capacity']

        # 简化评分
        score = soh * 0.6 + (capacity / 100) * 0.4

        return score

    def estimate_recycling_value(self, battery_data):
        """
        估算回收价值
        材料价格:
          - Li: ¥100/kg
          - Co: ¥300/kg
          - Ni: ¥150/kg
          - Mn: ¥30/kg
        """
        # 简化计算（假设NCM523）
        capacity = battery_data['capacity']  # Ah
        voltage = 3.7  # V
        energy = capacity * voltage / 1000  # kWh

        # 材料回收率
        recovery_rate = 0.95

        # 材料价格（简化）
        material_price = 200  # 元/kWh

        recycling_value = energy * material_price * recovery_rate

        return recycling_value
```

### 多准则决策分析

```python
class MCDADecisionMaker:
    def __init__(self):
        # 评估准则
        self.criteria = {
            'economic': 0.4,      # 经济性
            'technical': 0.3,     # 技术可行性
            'environmental': 0.2, # 环保性
            'market': 0.1         # 市场需求
        }

    def multi_criteria_analysis(self, battery_data):
        """
        多准则决策分析
        """
        alternatives = {
            'continue': self.evaluate_continue_use(battery_data),
            'cascade': self.evaluate_cascade_use(battery_data),
            'recycle': self.evaluate_recycling(battery_data)
        }

        # 加权评分
        scores = {}
        for alt, values in alternatives.items():
            scores[alt] = sum([
                values[criterion] * weight
                for criterion, weight in self.criteria.items()
            ])

        # 选择得分最高的方案
        best_alternative = max(scores, key=scores.get)

        return {
            'decision': best_alternative,
            'scores': scores,
            'confidence': scores[best_alternative]
        }

    def evaluate_continue_use(self, battery_data):
        """评估继续使用"""
        return {
            'economic': 0.9 if battery_data['soh'] > 0.8 else 0.5,
            'technical': 0.95,
            'environmental': 0.7,
            'market': 0.8
        }

    def evaluate_cascade_use(self, battery_data):
        """评估梯次利用"""
        soh = battery_data['soh']
        if 0.6 <= soh <= 0.8:
            return {
                'economic': 0.7,
                'technical': 0.8,
                'environmental': 0.9,
                'market': 0.6
            }
        else:
            return {
                'economic': 0.3,
                'technical': 0.4,
                'environmental': 0.8,
                'market': 0.4
            }

    def evaluate_recycling(self, battery_data):
        """评估回收"""
        return {
            'economic': 0.6,
            'technical': 0.9,
            'environmental': 0.95,
            'market': 0.7
        }
```

## 技术特点

### 1. 决策树算法
- **SOH阈值**: 80%（继续使用）、60%（梯次利用）
- **多因素评估**: 容量、内阻、循环寿命
- **价值最大化**: 选择收益最高的方案

### 2. 梯次利用评估
- **适用场景**: 储能系统、通信基站、低速电动车
- **技术要求**: 模组重组、BMS适配
- **市场前景**: 需求旺盛，价格稳定

### 3. 材料回收
- **回收技术**: 湿法冶金、火法冶金
- **回收率**: 锂85%、钴95%、镍90%
- **环保效益**: 减少 Mining，降低碳排放
