## 层级概述

L2 BMS层级是ASGARD平台的核心层级，负责电池管理系统板端的实时状态估计、安全监测和性能控制。该层级所有Skills均针对嵌入式环境优化，算力需求<50MHz，可在ARM Cortex-M4/M7或RISC-V处理器上运行。

### 核心目标

- **高精度状态估计**: SOC误差<±1.5%，SOH误差<±2%
- **实时安全监测**: 响应时间<100ms
- **智能均衡**: 一致性提升15%
- **低功耗**: 算法功耗<100mW

## 使用示例

### 1. 标准BMS配置（推荐）

```python
# SOC估计: AEKF (精度±1.5%)
soc_estimator = B2_4_AEKF()

# SOH估计: 两点法 (精度±5%)
soh_estimator = B2_6_TwoPoint()

# 安全监测
safety_monitor = B2_9_ExpansionForce()  # 全体系
# 或
safety_monitor = B2_10_LithiumPlating()  # 锂电池

# 性能控制
balancing = B2_11_IntelligentBalancing()
thermal_mgmt = B2_12_MPC()
safety_ctrl = B2_13_ActiveSafety()
```

### 2. 高精度BMS配置

```python
# SOC估计: UKF (精度±1%)
soc_estimator = B2_5_UKF()

# SOH估计: AEKF联合估计 (精度±1.5%)
soh_estimator = B2_7_AEKF_SOH()

# 其他同上
```

### 3. 钠离子电池专用配置

```python
# SOC估计: 分段滤波
soc_estimator = B2_14_SodiumSegmentedFilter()

# 其他通用算法
```

## 依赖关系

```

├─ 依赖: L0 基础模型层
│   ├─ B2.3-B2.5 依赖: F0.43 (牛顿法-优化)
│   ├─ B2.6 依赖: F0.1 (PCA-降维)
│   ├─ B2.11 依赖: F0.37 (二次规划)
│   └─ B2.12 依赖: F0.37 (二次规划)
└─ 输出到: L3 车载层级
    ├─ SOC/SOH → L3.1-L3.3 车辆状态估计
    └─ SOP → L3.4 能量管理
```

## 关键技术亮点

1. **自适应滤波**: B2.4 AEKF实时调整噪声协方差，适应老化
2. **高精度UKF**: B2.5在强非线性场景下精度达±1%
3. **钠离子专用**: B2.14针对硬碳平台区设计分段滤波
4. **多级安全**: B2.13提供L0-L4分级响应，避免误触发
