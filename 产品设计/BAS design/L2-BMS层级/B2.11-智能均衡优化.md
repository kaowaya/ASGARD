## 算法原理

### 均衡目标函数

$$
\min J = w_1 \cdot E_{loss} + w_2 \cdot S_{loss} + w_3 \cdot \Delta SOC
$$

其中:
- $E_{loss}$: 均衡能耗
- $S_{loss}$: 开关损耗（寿命）
- $\Delta SOC$: SOC不一致度

### 均衡策略

```python
def intelligent_balancing(soc_array, voltage_array, temp_array):
    """
    智能均衡决策
    Returns:
        actions: 每个电芯的均衡动作（0/1）
    """
    n_cells = len(soc_array)
    actions = np.zeros(n_cells, dtype=int)

    # 计算不一致度
    soc_mean = np.mean(soc_array)
    soc_std = np.std(soc_array)

    if soc_std < 0.02:  # 一致性良好，不均衡
        return actions

    # 找出需要放电的电芯（SOC > 平均）
    for i in range(n_cells):
        if soc_array[i] > soc_mean + 0.03 and temp_array[i] < 50:
            actions[i] = 1  # 开启均衡

    return actions
```
