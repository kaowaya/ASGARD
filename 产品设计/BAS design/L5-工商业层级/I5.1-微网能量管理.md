## Skill概述

微网能量管理是工商业层级的核心Skill，通过混合整数线性规划（MILP）算法，实现对光伏、风电、柴油发电机、储能系统和电网等多能源的协调优化，在保证供电可靠性的前提下，实现经济成本和碳排放的最小化。

### 核心价值
- **多能源协同**: 优化调度可再生能源、储能和传统能源
- **削峰填谷**: 降低需量电费，提升用电经济性
- **碳排放优化**: 在优化目标中引入碳成本，促进绿色用能
- **可靠性保障**: 优先保障关键负载供电，提升微网韧性

### 应用场景
- 工业园区微网
- 商业综合体微网
- 偏远地区独立微网
- 海岛微网系统

### 优化模型

```python
import numpy as np
from scipy.optimize import minimize
import cvxpy as cp

class MicrogridEnergyManagement:
    def __init__(self, microgrid_config):
        """
        微网能量管理器
        Args:
            microgrid_config: {
                'pv_capacity': 光伏容量 (kW)
                'wind_capacity': 风电容量 (kW)
                'diesel_capacity': 柴发容量 (kW)
                'battery_capacity': 储能容量 (kWh)
                'battery_power': 储能功率 (kW)
                'critical_load': 关键负载曲线 (kW)
                'flexible_load': 可调负载曲线 (kW)
                'grid_limit': 电网交互限制 (kW)
            }
        """
        self.config = microgrid_config

    def optimize_day_ahead(self, pv_forecast, wind_forecast, price_forecast, carbon_factor):
        """
        日前优化
        Args:
            pv_forecast: 光伏发电预测（24小时）
            wind_forecast: 风电发电预测（24小时）
            price_forecast: 电价预测（24小时）
            carbon_factor: 碳排放因子（gCO2/kWh）
        Returns:
            schedule: 优化后的调度方案
        """
        N = 24  # 24小时

        # 决策变量
        P_pv = pv_forecast  # 光伏出力
        P_wind = wind_forecast  # 风电出力
        P_diesel = cp.Variable(N)  # 柴发出力
        P_battery = cp.Variable(N)  # 电池功率（正=充电，负=放电）
        P_grid = cp.Variable(N)  # 电网功率（正=购电，负=卖电）
        P_curtail = cp.Variable(N)  # 弃风弃光
        soc = cp.Variable(N+1)  # 电池SOC

        # 约束条件
        constraints = []

        # 1. 功率平衡
        for t in range(N):
            constraints.append(
                P_pv[t] + P_wind[t] + P_diesel[t] + P_grid[t] + P_battery[t] - P_curtail[t]
                == self.config['critical_load'][t] + self.config['flexible_load'][t]
            )

        # 2. 柴发约束
        constraints.append(P_diesel >= 0)
        constraints.append(P_diesel <= self.config['diesel_capacity'])

        # 3. 电池约束
        eta = 0.95  # 充放电效率
        for t in range(N):
            # SOC演化
            constraints.append(
                soc[t+1] == soc[t] - P_battery[t] * eta / self.config['battery_capacity']
            )
            # SOC范围
            constraints.append(soc[t] >= 0.1)
            constraints.append(soc[t] <= 0.9)
        # SOC初始和终值
        constraints.append(soc[0] == 0.5)
        constraints.append(soc[N] == 0.5)

        # 4. 电池功率约束
        constraints.append(P_battery >= -self.config['battery_power'])
        constraints.append(P_battery <= self.config['battery_power'])

        # 5. 电网交互约束
        constraints.append(P_grid >= -self.config['grid_limit'])
        constraints.append(P_grid <= self.config['grid_limit'])

        # 6. 弃风弃光约束
        constraints.append(P_curtail >= 0)
        constraints.append(P_curtail <= P_pv + P_wind)

        # 目标函数（多目标加权）
        # 1. 运行成本
        cost_diesel = cp.sum(P_diesel * 0.8)  # 柴发成本0.8元/kWh
        cost_grid = cp.sum(P_grid * price_forecast)
        cost_total = cost_diesel + cost_grid

        # 2. 碳排放成本
        carbon_cost = cp.sum(
            (P_diesel * 800 + P_grid * carbon_factor) / 1000  # 柴发800gCO2/kWh
        )

        # 3. 弃风弃光惩罚
        penalty_curtail = cp.sum(P_curtail) * 1.0  # 惩罚系数

        objective = cp.Minimize(cost_total + 0.1 * carbon_cost + penalty_curtail)

        # 求解
        prob = cp.Problem(objective, constraints)
        prob.solve(solver=cp.ECOS)

        # 解析结果
        schedule = {
            'diesel': P_diesel.value,
            'battery': P_battery.value,
            'grid': P_grid.value,
            'curtailment': P_curtail.value,
            'soc': soc.value,
            'total_cost': cost_total.value,
            'carbon_emission': carbon_cost.value * 1000  # kgCO2
        }

        return schedule

    def real_time_dispatch(self, t, pv_actual, wind_actual, load_actual, soc_current, price_current):
        """
        实时调度
        优先级: 可再生能源 > 电池 > 柴发 > 电网
        """
        # 净负荷
        net_load = load_actual - pv_actual - wind_actual

        if net_load > 0:  # 缺电
            if soc_current > 0.2:  # 电池有电
                P_battery = min(net_load, self.config['battery_power'])
                P_diesel = net_load - P_battery
                P_grid = 0
            else:  # 电池没电
                P_battery = 0
                P_diesel = min(net_load, self.config['diesel_capacity'])
                P_grid = net_load - P_diesel
        else:  # 盈电
            if soc_current < 0.8:  # 电池未满
                P_battery = max(net_load, -self.config['battery_power'])
                P_grid = 0
            else:  # 电池已满
                P_battery = 0
                # 卖给电网（如果电价高于成本）
                if price_current > 0.5:
                    P_grid = net_load
                else:
                    # 弃风弃光
                    P_grid = 0

        return {
            'battery': P_battery,
            'diesel': P_diesel,
            'grid': P_grid
        }
```

## 技术特点

### 1. 多时间尺度优化
- **日前优化**: 基于预测数据，制定24小时调度计划
- **实时调度**: 基于实际数据，进行分钟级功率调整
- **滚动优化**: 每小时更新预测，动态调整策略

### 2. 多目标协调
- **经济性**: 最小化运行成本（电费+燃料费）
- **可靠性**: 保障关键负载供电，最小化弃风弃光
- **环保性**: 降低碳排放，提升清洁能源占比

### 3. 约束处理
- 功率平衡约束（供需实时平衡）
- 电池SOC约束（10%-90%安全范围）
- 电网交互限制（避免过载）
- 柴发出力限制（避免低效运行）
