# F0.3 线性回归（Linear Regression）

## 算法概述

线性回归是最基础的监督学习算法，通过拟合线性关系来预测连续值。在电池管理系统中，线性回归主要用于：

- **SOH估计**：基于充电时间、内阻、ICA峰值等特征估计健康状态
- **RUL预测**：预测电池剩余使用寿命
- **容量预测**：预测电池当前可用容量
- **参数估计**：辨识电池模型参数

## 核心原理

### 普通最小二乘法（OLS）

**目标**：最小化平方误差

```
min_β ||y - Xβ||²

其中：
- X: 特征矩阵 (n_samples, n_features)
- y: 目标值 (n_samples,)
- β: 回归系数 (n_features,)
```

### Ridge回归（L2正则化）

**目标**：
```
min_β ||y - Xβ||² + α||β||²

优势：
- 防止过拟合
- 处理多重共线性
- 解稳定
```

### Lasso回归（L1正则化）

**目标**：
```
min_β ||y - Xβ||² + α||β||₁

优势：
- 特征选择（稀疏解）
- 可解释性强
- 自动降维
```

## 代码示例

```python
from sklearn.linear_model import LinearRegression, Ridge, Lasso
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np

def soh_linear_regression(features, soh_true, method='Ridge', alpha=1.0):
    """
    基于线性回归的SOH估计

    特征：
    - 充电时间
    - 内阻
    - ICA峰值位置
    - ICA峰值高度

    参数:
        features: 特征矩阵 (n_samples, n_features)
        soh_true: 真实SOH值 (n_samples,)
        method: 'OLS', 'Ridge', or 'Lasso'
        alpha: 正则化系数

    返回:
        soh_pred: 预测SOH
        importance: 特征重要性（回归系数）
        metrics: 评估指标
    """
    # 标准化
    scaler = StandardScaler()
    X_std = scaler.fit_transform(features)

    # 训练
    if method == 'OLS':
        model = LinearRegression()
    elif method == 'Ridge':
        model = Ridge(alpha=alpha)
    elif method == 'Lasso':
        model = Lasso(alpha=alpha)

    model.fit(X_std, soh_true)

    # 预测
    soh_pred = model.predict(X_std)

    # 特征重要性（回归系数）
    importance = model.coef_

    # 评估指标
    mse = mean_squared_error(soh_true, soh_pred)
    r2 = r2_score(soh_true, soh_pred)

    metrics = {
        'MSE': mse,
        'RMSE': np.sqrt(mse),
        'R2': r2,
        'MAE': np.mean(np.abs(soh_true - soh_pred))
    }

    return soh_pred, importance, metrics

# 应用示例：SOH预测
def predict_soh_example():
    """
    SOH预测示例
    """
    # 模拟数据
    n_samples = 1000
    n_features = 4

    # 特征：充电时间、内阻、ICA峰值位置、ICA峰值高度
    features = np.random.rand(n_samples, n_features)

    # 真实SOH（模拟）
    soh_true = 0.9 - 0.001 * features[:, 0] + 0.5 * features[:, 1] + 0.001 * np.random.randn(n_samples)

    # 训练模型
    soh_pred, importance, metrics = soh_linear_regression(
        features,
        soh_true,
        method='Ridge',
        alpha=1.0
    )

    print(f"RMSE: {metrics['RMSE']:.4f}")
    print(f"R²: {metrics['R2']:.4f}")
    print(f"特征重要性: {importance}")

    return soh_pred, importance, metrics

# 应用：RUL预测
def rul_linear_regression(cycle_count, capacity_fade_rate, avg_voltage):
    """
    基于线性回归的RUL预测

    参数:
        cycle_count: 循环次数
        capacity_fade_rate: 容量衰减率
        avg_voltage: 平均电压

    返回:
        rul: 剩余寿命（循环次数）
    """
    # 特征构造
    features = np.column_stack([cycle_count, capacity_fade_rate, avg_voltage])

    # 假设已训练的模型（简化示例）
    # RUL = (0.8 - current_capacity) / fade_rate
    current_capacity = 1.0 - capacity_fade_rate * cycle_count

    if current_capacity <= 0.8:
        rul = 0
    else:
        rul = (current_capacity - 0.8) / capacity_fade_rate

    return int(rul)

# 特征重要性分析
def analyze_feature_importance(model, feature_names):
    """
    分析特征重要性

    参数:
        model: 训练好的回归模型
        feature_names: 特征名称列表

    返回:
        importance_df: 特征重要性DataFrame
    """
    import pandas as pd

    importance = model.coef_
    importance_df = pd.DataFrame({
        'feature': feature_names,
        'coefficient': importance,
        'abs_coefficient': np.abs(importance)
    })

    # 按绝对值排序
    importance_df = importance_df.sort_values('abs_coefficient', ascending=False)

    return importance_df
```

输入：充电时间、内阻、ICA峰值位置、ICA峰值高度

## 部署建议

### 云端部署
- **适用场景**：离线分析、模型训练
- **优势**：可处理大规模数据，支持交叉验证

### 边缘端部署
- **适用场景**：实时SOH估计
- **推荐算法**：Ridge回归（数值稳定）
- **优化方案**：
  - 预训练模型，仅做推理
  - 使用增量学习更新模型
  - 量化回归系数

## 性能优化

1. **特征工程**：
   - 标准化：消除量纲影响
   - 特征交叉：捕捉非线性关系
   - 特征选择：去除冗余特征

2. **正则化调优**：
   - Ridge：α ∈ [0.01, 100]
   - Lasso：α ∈ [0.001, 10]
   - 使用交叉验证选择最优α

3. **模型改进**：
   - 多项式特征：捕捉非线性
   - 特征交互：增加交互项
   - 集成学习：结合多个模型

## 评估指标

- **MSE**（均方误差）：越小越好
- **RMSE**（均方根误差）：越小越好
- **MAE**（平均绝对误差）：越小越好
- **R²**（决定系数）：0-1，越接近1越好
- **调整R²**：考虑特征数量
