## Skill概述

虚拟电站（Virtual Power Plant, VPP）通过分布式优化算法，将分散的储能、EV、可调负载等资源聚合为虚拟电厂，参与电网调频、调峰和电力市场交易，实现资源的高效利用和价值最大化。

### 核心价值
- **资源聚合**: 聚合海量分布式资源，形成规模效应
- **市场参与**: 参与调频、容量市场，获取收益
- **电网支撑**: 提供辅助服务，提升电网稳定性
- **去中心化**: 基于区块链的透明结算机制

### 应用场景
- 家庭储能聚合
- 工业用户响应
- EV车队管理
- 商业建筑虚拟电厂

### 分布式优化算法

```python
class VirtualPowerPlant:
    def __init__(self, resources):
        """
        虚拟电站
        Args:
            resources: 资源列表 [储能, EV, 负载...]
        """
        self.resources = resources

    def aggregate(self, demand_signal):
        """
        聚合资源响应电网需求
        Args:
            demand_signal: 需求信号 {功率, 持续时间}
        Returns:
            allocation: 各资源的功率分配
        """
        # 1. 评估各资源可用性
        available_power = []
        for resource in self.resources:
            power = resource.get_available_power()
            available_power.append(power)

        total_power = sum(available_power)

        # 2. 分配策略
        if total_power >= demand_signal['power']:
            # 按比例分配
            allocation = [
                demand_signal['power'] * power / total_power
                for power in available_power
            ]
        else:
            # 全部分配
            allocation = available_power

        # 3. 下发指令
        for i, resource in enumerate(self.resources):
            resource.dispatch(allocation[i])

        return allocation

    def market_bidding(self, price_forecast):
        """
        市场竞价策略
        """
        # 预测未来市场价格
        high_price_hours = price_forecast > np.percentile(price_forecast, 80)

        # 在高电价时段预留容量
        reserved_capacity = 0
        for resource in self.resources:
            if resource.soc > 0.5:  # SOC > 50%
                reserved_capacity += resource.capacity * (resource.soc - 0.5)

        # 竞价策略
        bid_price = np.mean(price_forecast[high_price_hours]) * 0.9  # 略低于预测价格

        return {
            'capacity': reserved_capacity,
            'price': bid_price,
            'hours': high_price_hours
        }
```

### 区块链结算机制

```python
class BlockchainSettlement:
    def __init__(self):
        """
        基于区块链的结算系统
        """
        self.chain = []
        self.pending_transactions = []

    def create_transaction(self, from_address, to_address, amount, data):
        """
        创建交易
        Args:
            from_address: 付款方（聚合商/电网）
            to_address: 收款方（资源提供商）
            amount: 金额
            data: 交易数据（提供的服务、功率、时间等）
        """
        transaction = {
            'from': from_address,
            'to': to_address,
            'amount': amount,
            'data': data,
            'timestamp': time.time(),
            'hash': self.calculate_hash({
                'from': from_address,
                'to': to_address,
                'amount': amount,
                'timestamp': time.time()
            })
        }
        self.pending_transactions.append(transaction)
        return transaction

    def settle(self, block_reward=0):
        """
        结算区块
        """
        if not self.pending_transactions:
            return None

        block = {
            'index': len(self.chain) + 1,
            'timestamp': time.time(),
            'transactions': self.pending_transactions,
            'previous_hash': self.chain[-1]['hash'] if self.chain else '0',
            'nonce': 0
        }

        # 挖矿（简化版）
        block['hash'] = self.calculate_hash(block)
        self.chain.append(block)
        self.pending_transactions = []

        return block

    def calculate_hash(self, data):
        """
        计算哈希（简化）
        """
        import hashlib
        data_str = str(data).encode('utf-8')
        return hashlib.sha256(data_str).hexdigest()
```

## 技术特点

### 1. 分布式资源聚合
- **异构资源**: 支持储能、EV、负载等多种资源
- **动态聚合**: 实时评估资源可用性
- **智能分配**: 按比例或优先级分配功率

### 2. 市场竞价策略
- **价格预测**: 基于历史数据和机器学习
- **容量预留**: 在高电价时段预留容量
- **竞价优化**: 最大化收益的同时保证中标率

### 3. 区块链结算
- **透明性**: 所有交易公开可查
- **不可篡改**: 防止欺诈和纠纷
- **自动执行**: 智能合约自动结算
