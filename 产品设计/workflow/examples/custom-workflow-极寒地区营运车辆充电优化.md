# Custom Workflow示例：极寒地区营运车辆充电优化

> 版本：V1.0
> 基于Meta-Workflow：析锂延寿Workflow
> 客户：东北某出租车公司
> 电池类型：NCM-811
> 环境温度：-30°C ~ -10°C

---

## 需求理解

### 客户描述

"我是东北出租营运车辆，冬天充电太慢，电池也不耐用。"

### 需求分析

| 维度 | 分析结果 |
|-----|---------|
| **应用场景** | 低温快充 |
| **电池类型** | NCM-811三元锂 |
| **环境条件** | 极寒（-30°C ~ -10°C） |
| **核心痛点** | 充电慢 + 容量衰减快 |
| **业务目标** | 充电效率↑30% + 寿命延长↑20% |

## Workflow生成流程

### Step 1: Meta-Workflow匹配

```
用户需求："东北出租营运车辆，冬天充电慢，电池不耐用"
       ↓
语义分析：
  - 关键词：营运车辆、充电、低温、寿命
  - 意图：充电优化 + 析锂抑制
       ↓
Meta-Workflow匹配：
  - 析锂延寿Workflow（相似度：85%）
  - 热失控预防Workflow（相似度：30%）
  - 储能电站健康管理Workflow（相似度：20%）
       ↓
选择：析锂延寿Workflow（基础模板）
```

---

### Step 2: 个性化分析

#### 环境分析
- **温度范围**：-30°C ~ -10°C（极端低温）
- **温度变化**：日温差>15°C
- **充电场景**：户外快充站，无预热条件

#### 营运车辆分析
- **车队规模**：500辆出租车
- **日均充电**：2次/车
- **充电时长**：2-3小时（优化前）
- **可用率要求**：>90%

#### 数据分析
- **历史充电曲线**：ICA主峰左移50mV（析锂特征）
- **容量衰减率**：8%/月（冬季）
- **析锂发生率**：45%

---

### Step 3: BAS微调

#### 新增BAS

| BAS | 用途 | 理由 |
|-----|------|------|
| **气象数据接口** | 获取环境温度预测 | 提前预热电池 |
| **B2.8 钠电分段滤波** | 支持混合营运车辆 | 部分车辆使用钠电池 |

#### 参数调整

| BAS | 原始参数 | 微调后参数 | 调优理由 |
|-----|---------|-----------|---------|
| **B2.5** | Φanode < 0V | Φanode < -0.05V | 低温环境容差放宽 |
| **B2.1** | MCC: 0.3C/0.5C/0.7C | 0.2C/0.3C/0.5C | 低温保守策略 |
| **B2.6** | MPC时域30s | MPC时域60s | 低温热惯性大 |
| **C3.7** | 峰值窗口3.6-4.0V | 峰值窗口3.4-4.0V | 低温电压平台偏移 |

#### 强化BAS

| BAS | 强化方向 | 具体措施 |
|-----|---------|---------|
| **C3.7 ICA分析** | 低温特征提取 | 增加低温段特征权重 |

## 智能充电策略

### 策略表

| 温度范围 | 析锂风险 | 充电电流 | 冷却策略 | 预热策略 |
|---------|---------|---------|---------|---------|
| T > -10°C | 低 | 0.5C | 自然冷却 | 无 |
| -20°C < T ≤ -10°C | 中 | 0.3C | 弱冷却 | 预热至-10°C |
| -30°C < T ≤ -20°C | 高 | 0.2C | 强冷却 | 预热至-15°C |
| T ≤ -30°C | 极高 | 禁止快充 | - | 预热至-20°C |

### 充电曲线

```
电流 (C)
 0.7 |        ┌────────────
     |       /
 0.5 |      /
     |     /
 0.3 |    /
     |   /
 0.2 |  /
     | /
 0.1 |/
     └──────────────────────→ SOC (%)
     0  20  40  60  80  100

-20°C时充电策略：
  - SOC 0-20%: 0.2C（涓流预热）
  - SOC 20-80%: 0.3C（恒流）
  - SOC 80-100%: 0.2C（降流）
```

## 复制与推广

### 适用场景

- ✓ 出租车/网约车（北方地区）
- ✓ 物流车（低温配送）
- ✓ 电动公交（冬季运行）
- ✓ 私家车（快充场景）

### 推广建议

1. **电池类型**：NCM/NCA三元锂（LFP低温性能更好，可适当放宽策略）
2. **温度范围**：-20°C ~ -10°C（可直接复制），-30°C需进一步优化
3. **充电功率**：快充站（>60kW），慢充站效果不明显
4. **数据积累**：需至少1个月的历史数据用于参数调优

---

## 技术总结

### 关键创新点

1. **气象数据融合**：提前预热，降低析锂风险
2. **温度自适应**：分段策略，平衡速度与安全
3. **脉冲修复**：反向抑制析锂，延长电池寿命
4. **持续优化**：数据飞轮，越用越好

### 最佳实践

- ✓ 充分利用Meta-Workflow模板，降低开发成本
- ✓ 基于客户实际数据微调，提升适配性
- ✓ 渐进式优化，先上线核心功能，再迭代增强
- ✓ 建立监控体系，持续跟踪效果
