# F0.13 K近邻算法（K-Nearest Neighbors）

## 算法概述

KNN是一种简单的监督学习算法，基于"近朱者赤，近墨者黑"的思想。在电池管理系统中，KNN可用于：

- **分类任务**：电池状态分类
- **回归任务**：SOH估计
- **异常检测**：基于距离的异常识别

## 核心原理

**核心思想**：找到最近的k个样本，投票或平均

**分类**：多数投票
**回归**：k个邻居的平均值

**距离度量**：
- 欧氏距离
- 曼哈顿距离
- 余弦相似度

## 代码示例

```python
from sklearn.neighbors import KNeighborsClassifier, KNeighborsRegressor

def knn_soh_estimation(X_train, y_train, X_test, k=5):
    """
    KNN回归估计SOH
    """
    knn = KNeighborsRegressor(n_neighbors=k)
    knn.fit(X_train, y_train)
    soh_pred = knn.predict(X_test)
    return soh_pred
```

## 应用场景

- **SOH估计**：相似电池的SOH
- **状态分类**：工作状态识别
- **推荐系统**：相似电池推荐
