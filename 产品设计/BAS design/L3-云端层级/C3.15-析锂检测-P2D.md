## 算法原理

### P2D模型负极电位追踪

通过P2D模型计算负极电位，当负极电位低于0V（vs Li/Li+）时，发生析锂。

$$
\phi_s^{neg} < 0 \Rightarrow \text{Lithium Plating}
$$

## 代码示例

### Python实现

```python
import numpy as np

class P2D_LithiumPlating:
    def __init__(self, params):
        """
        P2D模型参数初始化
        params: {
            'D_s_neg': 1e-14,  # 负极固相扩散系数
            'D_s_pos': 1e-14,  # 正极固相扩散系数
            'k_neg': 1e-11,    # 负极反应速率常数
            'k_pos': 1e-11,    # 正极反应速率常数
            'epsilon_neg': 0.4, # 负极孔隙率
            'epsilon_pos': 0.3  # 正极孔隙率
        }
        """
        self.D_s_neg = params['D_s_neg']
        self.D_s_pos = params['D_s_pos']
        self.k_neg = params['k_neg']
        self.k_pos = params['k_pos']
        self.epsilon_neg = params['epsilon_neg']
        self.epsilon_pos = params['epsilon_pos']

    def calculate_anode_potential(self, I_app, SOC, T=298):
        """
        计算负极电位
        Args:
            I_app: 施加电流（A/m²）
            SOC: 荷电状态
            T: 温度（K）
        Returns:
            phi_s_neg: 负极电位（V vs Li/Li+）
        """
        # 简化模型：负极电位 = OCV(SOC) - 过电位
        OCV_neg = self.get_anode_ocv(SOC)

        # 过电位计算（Butler-Volmer方程简化）
        eta_neg = (8.314 * T / (0.5 * 96485)) * np.arcsinh(I_app / (2 * self.k_neg))

        phi_s_neg = OCV_neg - eta_neg

        return phi_s_neg

    def get_anode_ocv(self, SOC):
        """
        获取负极OCV
        简化模型：石墨负极OCV
        """
        # 简化OCV-SOC关系（实际需要查表）
        if SOC > 0.9:
            return 0.1
        elif SOC > 0.5:
            return 0.05
        else:
            return 0.0

    def detect_lithium_plating(self, I_app, SOC, T=298):
        """
        检测析锂
        Args:
            I_app: 施加电流（A/m²）
            SOC: 荷电状态
            T: 温度（K）
        Returns:
            is_plating: 是否析锂
            phi_s_neg: 负极电位
        """
        phi_s_neg = self.calculate_anode_potential(I_app, SOC, T)

        # 析锂判据：负极电位 < 0V
        is_plating = phi_s_neg < 0

        return is_plating, phi_s_neg

    def calculate_plating_current(self, I_app, SOC, T=298):
        """
        计算析锂电流
        Args:
            I_app: 施加电流（A/m²）
            SOC: 荷电状态
            T: 温度（K）
        Returns:
            I_plating: 析锂电流（A/m²）
        """
        phi_s_neg = self.calculate_anode_potential(I_app, SOC, T)

        # 如果负极电位 < 0V，发生析锂
        if phi_s_neg < 0:
            # 析锂电流（简化的Tafel方程）
            I_plating = -self.k_neg * np.exp(-0.5 * 96485 * phi_s_neg / (8.314 * T))
        else:
            I_plating = 0

        return I_plating
```

## 技术特点

### 优势

1. **物理机理清晰**: 基于电化学第一性原理
2. **高精度**: 可准确预测析锂发生
3. **定量分析**: 可计算析锂电流
4. **设计优化**: 可用于充电策略优化

### 局限性

1. **计算复杂**: 需要求解偏微分方程
2. **参数需求多**: 需要大量电化学参数
3. **仅适用于锂电池**: 不适用于钠电池等
4. **实时性差**: 计算时间长，不适合实时检测
