## 算法原理

### 1. EIS等效电路模型

```
电路模型: R(QR)(QRW)

阻抗表达式:
  Z(ω) = R_s + \frac{1}{jωC_{dl} + \frac{1}{R_{ct} + Z_W}}

  其中:
    R_s: 欧姆内阻
    C_{dl}: 双电层电容
    R_{ct}: 电荷转移电阻
    Z_W: Warburg阻抗（扩散）

Warburg阻抗:
  Z_W = σ_W(1-j)/√ω

  其中σ_W与扩散系数D相关:
    σ_W ∝ 1/√D
```

### 2. 注液量-扩散系数关系

```
假设: 注液量不足 → 扩散系数D↓ → σ_W↑

标定曲线:
  D(注液量) = D_0 · (1 - exp(-k·V_injected))

优化目标:
  min ||Z_EIS_measured - Z_EC_model(θ)||²

  其中θ = [R_s, C_{dl}, R_{ct}, σ_W]
```

## 代码示例

```python
import numpy as np
from impedance.models.circuits import CustomCircuit

# 定义等效电路
circuit = 'R0-p(R1,C1)-W1'
model = CustomCircuit(circuit, initial_guess=[0.01, 0.01, 1000, 0.1])

# 拟合EIS数据
frequencies = np.logspace(-2, 5, 50)  # 0.01 Hz - 100 kHz
Z_measured = ...  # 测量阻抗

model.fit(frequencies, Z_measured)
print(f"拟合参数: {model.parameters_}")

# 计算扩散系数
sigma_W = model.parameters_[3]
D_estimated = 1 / (sigma_W ** 2)  # 简化关系

# 注液量推荐
def recommend_injection(D_target, D_current, current_injection):
    """
    根据扩散系数调整注液量
    """
    k = 0.01  # 标定系数
    D_0 = 1e-10  # 参考扩散系数

    # 反推目标注液量
    V_target = -np.log(1 - D_target/D_0) / k
    adjustment = V_target - current_injection

    return adjustment

adjustment = recommend_injection(D_0, D_estimated, current_injection=5.0)
print(f"建议调整注液量: {adjustment:.2f} g")
```
