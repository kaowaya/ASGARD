# F0.5 随机森林（Random Forest）

## 算法概述

随机森林是一种基于Bagging的集成学习算法，通过构建多个决策树并聚合预测结果来提高性能。在电池管理系统中，随机森林主要用于：

- **SOH/RUL预测**：高精度回归预测
- **特征重要性排序**：识别关键影响因素
- **故障分类**：多类别分类任务

## 核心原理

**核心思想**：Bagging + 决策树

**步骤**：
1. Bootstrap采样：有放回抽样生成k个训练集
2. 构建决策树：每个树基于随机特征子集
3. 聚合预测：
   - 分类：多数投票
   - 回归：平均

**优势**：
- 鲁棒性强（降低方差）
- 可处理缺失值
- 提供特征重要性
- 抗过拟合

## 代码示例

```python
from sklearn.ensemble import RandomForestRegressor, RandomForestClassifier
from sklearn.metrics import mean_squared_error, accuracy_score
import numpy as np
import pandas as pd

def soh_random_forest(X_train, y_train, X_test):
    """
    基于随机森林的SOH估计
    """
    model = RandomForestRegressor(
        n_estimators=100,    # 树的数量
        max_depth=10,        # 最大深度
        min_samples_split=5, # 最小分割样本数
        min_samples_leaf=2,  # 叶节点最小样本数
        random_state=42,
        n_jobs=-1           # 并行
    )
    model.fit(X_train, y_train)

    # 预测
    soh_pred = model.predict(X_test)

    # 特征重要性
    importance = model.feature_importances_

    return soh_pred, importance

def analyze_feature_importance(model, feature_names):
    """
    分析特征重要性
    """
    importance_df = pd.DataFrame({
        'feature': feature_names,
        'importance': model.feature_importances_
    }).sort_values('importance', ascending=False)

    return importance_df
```
