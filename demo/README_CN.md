# ASGARD 产品Demo

[English](README.md) | [简体中文]

> ASGARD核心能力前端演示：Agent智能编排、BAS技能库和智能数据分析。

## 快速开始

直接用浏览器打开 `index.html` 即可，无需服务器。

## 核心功能

- **Agent智能编排**：对话式AI工作流设计界面
- **DAG可视化**：实时工作流可视化展示
- **BAS技能库**：94+电池分析技能
- **数据分析**：交互式ECharts图表

## 技术栈

- HTML5
- JavaScript ES6+
- Tailwind CSS (CDN)
- ECharts (CDN)
- Font Awesome (CDN)

## 开发

```bash
# 使用Python启动简单HTTP服务器（可选）
python -m http.server 8000
```

## 浏览器兼容性

- Chrome/Edge 90+
- Safari 14+
- Firefox 88+
- 移动端Safari iOS 14+
- 移动端Chrome Android 10+
