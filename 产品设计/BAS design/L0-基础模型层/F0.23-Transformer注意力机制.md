# F0.23 Transformer注意力机制

## 算法概述

Transformer是一种基于自注意力机制的架构，完全摒弃了循环结构。在电池管理系统中，Transformer主要用于：

- **长序列RUL预测**：超长时序建模
- **多变量时序预测**：V、I、T多变量联合预测
- **异常检测**：注意力权重可视化
- **特征提取**：序列表示学习

## Self-Attention公式

**缩放点积注意力**：
```
Attention(Q, K, V) = softmax(Q·K^T / √d_k) · V

其中：
- Q (Query): 查询矩阵
- K (Key): 键矩阵
- V (Value): 值矩阵
- d_k: 键的维度
```

**多头注意力**：
```
MultiHead(Q, K, V) = Concat(head₁, ..., head_h) · W^O

其中：
head_i = Attention(Q·W_i^Q, K·W_i^K, V·W_i^V)
```

**优势**：
- 并行计算（比RNN快）
- 长距离依赖（比LSTM好）
- 可解释性（注意力权重可视化）

## 架构图

```
Input → Positional Encoding → Multi-Head Attention
      → Add & Norm → Feed-Forward → Add & Norm → Output
```

## 代码示例

```python
import torch
import torch.nn as nn
import math

class MultiHeadAttention(nn.Module):
    """
    多头注意力机制
    """
    def __init__(self, d_model, n_heads):
        super().__init__()
        assert d_model % n_heads == 0

        self.d_model = d_model
        self.n_heads = n_heads
        self.d_k = d_model // n_heads

        # Q, K, V线性变换
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)

        # 输出线性变换
        self.W_o = nn.Linear(d_model, d_model)

    def forward(self, query, key, value, mask=None):
        batch_size = query.size(0)

        # 线性变换
        Q = self.W_q(query)
        K = self.W_k(key)
        V = self.W_v(value)

        # 分割为多头
        Q = Q.view(batch_size, -1, self.n_heads, self.d_k).transpose(1, 2)
        K = K.view(batch_size, -1, self.n_heads, self.d_k).transpose(1, 2)
        V = V.view(batch_size, -1, self.n_heads, self.d_k).transpose(1, 2)

        # 计算注意力
        scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.d_k)

        if mask is not None:
            scores = scores.masked_fill(mask == 0, -1e9)

        attn_weights = torch.softmax(scores, dim=-1)
        context = torch.matmul(attn_weights, V)

        # 合并多头
        context = context.transpose(1, 2).contiguous().view(batch_size, -1, self.d_model)

        # 输出变换
        output = self.W_o(context)

        return output, attn_weights

class BatteryTransformer(nn.Module):
    """
    基于Transformer的电池时序预测
    """
    def __init__(self, input_dim, d_model, n_heads, n_layers, output_dim):
        super().__init__()

        # 输入嵌入
        self.embedding = nn.Linear(input_dim, d_model)

        # Positional Encoding
        self.pos_encoding = PositionalEncoding(d_model)

        # Transformer Encoder
        encoder_layer = nn.TransformerEncoderLayer(
            d_model=d_model,
            nhead=n_heads,
            dim_feedforward=d_model * 4,
            dropout=0.1
        )
        self.transformer = nn.TransformerEncoder(encoder_layer, num_layers=n_layers)

        # 输出层
        self.fc = nn.Linear(d_model, output_dim)

    def forward(self, x):
        # x: (batch, seq_len, input_dim)
        x = self.embedding(x)
        x = self.pos_encoding(x)

        # Transformer: (seq_len, batch, d_model)
        x = x.transpose(0, 1)
        x = self.transformer(x)

        # 取最后一个时间步
        x = x[-1]  # (batch, d_model)

        # 预测
        output = self.fc(x)

        return output

class PositionalEncoding(nn.Module):
    """
    位置编码
    """
    def __init__(self, d_model, max_len=5000):
        super().__init__()

        pe = torch.zeros(max_len, d_model)
        position = torch.arange(0, max_len, dtype=torch.float).unsqueeze(1)
        div_term = torch.exp(torch.arange(0, d_model, 2).float() * (-math.log(10000.0) / d_model))

        pe[:, 0::2] = torch.sin(position * div_term)
        pe[:, 1::2] = torch.cos(position * div_term)

        self.register_buffer('pe', pe.unsqueeze(0))

    def forward(self, x):
        # x: (batch, seq_len, d_model)
        x = x + self.pe[:, :x.size(1), :]
        return x
```

## 应用场景

### 场景1：长序列RUL预测
```
输入：1000个时间步的电压、电流、温度
处理：Transformer
输出：未来100步的容量衰减
价值：比LSTM更准确，计算更快
```

### 场景2：多变量时序预测
```
输入：电压、电流、温度、应力等多维时序
处理：Multi-Head Attention
输出：未来SOC、SOH、RUL
价值：捕捉变量间的复杂依赖关系
```
