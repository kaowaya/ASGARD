# F0.22 LSTM长短期记忆网络

## 算法概述

LSTM是一种特殊的RNN架构，通过门控机制解决长期依赖问题。在电池管理系统中，LSTM主要用于：

- **RUL预测**：剩余使用寿命预测
- **SOC轨迹预测**：未来SOC变化
- **SOH估计**：健康状态预测
- **负荷预测**：功率需求预测

## LSTM Cell公式

**遗忘门**：
```
f_t = σ(W_f · [h_{t-1}, x_t] + b_f)
```

**输入门**：
```
i_t = σ(W_i · [h_{t-1}, x_t] + b_i)
```

**候选值**：
```
ñ_t = tanh(W_ñ · [h_{t-1}, x_t] + b_ñ)
```

**更新细胞状态**：
```
C_t = f_t * C_{t-1} + i_t * ñ_t
```

**输出门**：
```
o_t = σ(W_o · [h_{t-1}, x_t] + b_o)
```

**隐藏状态**：
```
h_t = o_t * tanh(C_t)
```

## 架构对比

| 模型 | 特点 | 适用场景 |
|-----|------|---------|
| **RNN** | 简单，但梯度消失 | 短序列 |
| **LSTM** | 引入门控机制，学习长期依赖 | 长序列（推荐） |
| **GRU** | LSTM简化版，参数少 | 资源受限场景 |

## 代码示例

```python
import torch
import torch.nn as nn

class BatteryLSTM(nn.Module):
    """
    基于LSTM的RUL预测

    输入：历史充放电数据 (batch, seq_len, input_dim)
    输出：剩余寿命（RUL）
    """
    def __init__(self, input_dim, hidden_dim, n_layers=2):
        super().__init__()

        # LSTM层
        self.lstm = nn.LSTM(
            input_size=input_dim,
            hidden_size=hidden_dim,
            num_layers=n_layers,
            batch_first=True,
            dropout=0.2
        )

        # 全连接层
        self.fc = nn.Linear(hidden_dim, 1)

    def forward(self, x):
        # LSTM: (batch, seq_len, hidden_dim)
        lstm_out, (h_n, c_n) = self.lstm(x)

        # 取最后一个时间步的隐藏状态
        last_hidden = lstm_out[:, -1, :]

        # 预测RUL
        rul = self.fc(last_hidden)

        return rul

# 应用：RUL预测

class SOCPredictor(nn.Module):
    """
    SOC轨迹预测（多步预测）
    """
    def __init__(self, input_dim, hidden_dim, output_dim):
        super().__init__()

        self.lstm = nn.LSTM(input_dim, hidden_dim, batch_first=True)
        self.fc = nn.Linear(hidden_dim, output_dim)

    def forward(self, x, future_steps=10):
        """
        多步预测

        输入：
            x: (batch, seq_len, input_dim)
            future_steps: 预测未来多少步

        输出：
            soc_future: (batch, future_steps)
        """
        # LSTM编码历史
        lstm_out, (h_n, c_n) = self.lstm(x)

        # 自回归预测
        soc_future = []
        x_last = x[:, -1:, :]  # 最后一个时间步

        for _ in range(future_steps):
            # 预测下一步SOC
            lstm_out, (h_n, c_n) = self.lstm(x_last, (h_n, c_n))
            soc_next = self.fc(lstm_out)
            soc_future.append(soc_next)

            # 更新输入
            x_last = torch.cat([x_last, soc_next], dim=-1)

        soc_future = torch.cat(soc_future, dim=1)

        return soc_future
```
