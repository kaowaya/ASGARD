# F0.19 Bagging集成（Bootstrap Aggregating）

## 算法概述

Bagging是一种并行集成方法，通过Bootstrap采样降低模型方差。在电池管理系统中，Bagging可用于：

- **SOH估计**：稳定回归预测
- **模型稳定性**：降低过拟合
- **并行计算**：加速训练

## 核心原理

**核心思想**：降低方差

**步骤**：
1. Bootstrap采样：有放回抽样
2. 训练基学习器
3. 聚合预测：
   - 分类：多数投票
   - 回归：平均

**优势**：
- 降低方差
- 并行训练
- 防止过拟合

## 代码示例

```python
from sklearn.ensemble import BaggingRegressor
from sklearn.tree import DecisionTreeRegressor

def soh_bagging(X_train, y_train, X_test):
    """
    Bagging估计SOH
    """
    base = DecisionTreeRegressor()
    bagging = BaggingRegressor(
        base_estimator=base,
        n_estimators=50,
        max_samples=0.8,
        max_features=0.8,
        n_jobs=-1,
        random_state=42
    )
    bagging.fit(X_train, y_train)
    soh_pred = bagging.predict(X_test)
    return soh_pred, bagging
```

## 应用场景

- **SOH估计**：稳定预测
- **方差降低**：模型鲁棒性
- **并行训练**：加速计算
