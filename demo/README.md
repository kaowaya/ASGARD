# ASGARD 产品Demo

## 运行方式

直接用浏览器打开 `index.html` 即可，无需服务器。

## 技术栈

- HTML5
- JavaScript ES6+
- Tailwind CSS (CDN)
- ECharts (CDN)
- Font Awesome (CDN)

## 开发

```bash
# 使用Python启动简单HTTP服务器（可选）
python -m http.server 8000
```

## 浏览器兼容性

- Chrome/Edge 90+
- Safari 14+
- Firefox 88+
- 移动端Safari iOS 14+
- 移动端Chrome Android 10+
