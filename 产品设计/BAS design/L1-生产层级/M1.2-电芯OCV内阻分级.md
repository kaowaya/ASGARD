## 算法原理

### 1. DBSCAN密度聚类

```
核心思想: 基于密度发现任意形状的簇

参数:
  - eps: 邻域半径（OCV偏差<5mV，内阻偏差<0.1mΩ）
  - min_samples: 最小样本数（根据批次大小调整）

流程:
  1. 计算所有电芯的[OCV, 内阻]特征向量
  2. 对每个样本，统计eps邻域内的样本数
  3. 若≥min_samples，标记为核心点
  4. 将密度相连的核心点归为同一簇
  5. 标记噪声点（异常电芯）
```

### 2. Isolation Forest异常检测

```
原理: 异常样本更容易被孤立

算法流程:
  1. 随机选择特征和分割点
  2. 递归构建二叉树
  3. 计算路径长度（path length）
  4. 路径越短，越可能是异常

异常分数:
  s(x, n) = 2^{-E(h(x))/c(n)}

  其中:
    h(x): 样本x的路径长度
    c(n) = 2H(n-1) - (2(n-1)/n): 归一化因子
    H(k): 调和数
```

### 3. 分级策略

```python
def grade_cells(df, ocv_col='OCV', ir_col='IR'):
    """
    电芯分级
    Args:
        df: DataFrame, 包含OCV和内阻列
        ocv_col: OCV列名
        ir_col: 内阻列名
    Returns:
        grades: 分级标签 ['A', 'B', 'C', 'X']
    """
    from sklearn.preprocessing import StandardScaler
    from sklearn.cluster import DBSCAN
    from sklearn.ensemble import IsolationForest

    # 标准化
    scaler = StandardScaler()
    X = scaler.fit_transform(df[[ocv_col, ir_col]])

    # DBSCAN聚类
    dbscan = DBSCAN(eps=0.3, min_samples=10)
    clusters = dbscan.fit_predict(X)

    # Isolation Forest异常检测
    iso_forest = IsolationForest(contamination=0.05, random_state=42)
    outliers = iso_forest.fit_predict(X)

    # 分级逻辑
    grades = []
    for i, (cluster, outlier) in enumerate(zip(clusters, outliers)):
        if outlier == -1:
            grades.append('X')  # 异常电芯
        elif cluster == 0:
            # A级：OCV和内阻均最优
            if df[ocv_col].iloc[i] > df[ocv_col].median() and \
               df[ir_col].iloc[i] < df[ir_col].median():
                grades.append('A')
            else:
                grades.append('B')
        else:
            grades.append('C')

    return grades
```

## 代码示例

```python
import pandas as pd
import numpy as np
from sklearn.cluster import DBSCAN
from sklearn.ensemble import IsolationForest
import matplotlib.pyplot as plt

# 加载数据
df = pd.read_csv('cell_data.csv')  # 列: Cell_ID, OCV_V, IR_mOhm

# 分级
df['Grade'] = grade_cells(df)

# 统计
print(df['Grade'].value_counts())

# 可视化
fig, ax = plt.subplots(figsize=(10, 6))
colors = {'A': 'green', 'B': 'blue', 'C': 'orange', 'X': 'red'}
for grade in df['Grade'].unique():
    subset = df[df['Grade'] == grade]
    ax.scatter(subset['OCV_V'], subset['IR_mOhm'],
               c=colors[grade], label=grade, alpha=0.6)
ax.set_xlabel('OCV (V)')
ax.set_ylabel('Internal Resistance (mΩ)')
ax.set_title('Cell Grading: OCV vs IR')
ax.legend()
plt.grid(True, alpha=0.3)
plt.savefig('cell_grading.png', dpi=300)
```
