## 算法原理

### 1. One-Class SVM

```
目标: 学习正常化成曲线的边界，识别异常

目标函数:
  min 1/2||w||² + 1/νn Σξ_i - ρ

  s.t. w·φ(x_i) ≥ ρ - ξ_i
       ξ_i ≥ 0

其中:
  ν: 异常比例上限（0.01-0.1）
  ξ_i: 松弛变量
  ρ: 偏置项
```

### 2. ICA特征提取

```
目标: 分离化成曲线的独立成分

算法: FastICA

步骤:
  1. 中心化: x ← x - E[x]
  2. 白化: z = Vx, 使得Cov(z) = I
  3. 迭代: w ← E[zg(wᵀz)] - E[g'(wᵀz)]w
  4. 归一化: w ← w / ||w||
  5. 重复3-4直到收敛

非线性函数g:
  g(u) = tanh(u)  # 常用选择
```

## 代码示例

```python
from sklearn.svm import OneClassSVM
from sklearn.decomposition import FastICA
import numpy as np

# 加载正常化成曲线数据
X_normal = np.load('normal_formation_curves.npy')  # shape: (n_samples, n_timesteps)

# ICA降维
ica = FastICA(n_components=10, random_state=42)
X_ica = ica.fit_transform(X_normal)

# 训练One-Class SVM
clf = OneClassSVM(nu=0.05, kernel='rbf', gamma='scale')
clf.fit(X_ica)

# 检测新曲线
X_new = np.load('new_formation_curve.npy').reshape(1, -1)
X_new_ica = ica.transform(X_new)
prediction = clf.predict(X_new_ica)  # 1: 正常, -1: 异常

if prediction[0] == -1:
    print("⚠️ 检测到异常化成曲线!")
```
