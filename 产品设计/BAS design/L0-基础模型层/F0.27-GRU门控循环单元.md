# F0.27 GRU门控循环单元

## 算法概述

GRU是LSTM的简化版本，参数更少，训练更快。在电池管理系统中，GRU是LSTM的理想替代，特别适合资源受限场景。

## GRU公式

**更新门**：
```
z_t = σ(W_z · [h_{t-1}, x_t])
```

**重置门**：
```
r_t = σ(W_r · [h_{t-1}, x_t])
```

**候选隐藏状态**：
```
ĥ_t = tanh(W · [r_t * h_{t-1}, x_t])
```

**最终隐藏状态**：
```
h_t = (1 - z_t) * h_{t-1} + z_t * ĥ_t
```

## 代码示例

```python
import torch
import torch.nn as nn

class BatteryGRU(nn.Module):
    """
    基于GRU的电池时序预测
    """
    def __init__(self, input_dim, hidden_dim, n_layers=2):
        super().__init__()

        self.gru = nn.GRU(
            input_size=input_dim,
            hidden_size=hidden_dim,
            num_layers=n_layers,
            batch_first=True,
            dropout=0.2
        )

        self.fc = nn.Linear(hidden_dim, 1)

    def forward(self, x):
        # GRU: (batch, seq_len, hidden_dim)
        gru_out, h_n = self.gru(x)

        # 取最后一个时间步
        last_hidden = gru_out[:, -1, :]

        # 预测
        output = self.fc(last_hidden)

        return output
```
