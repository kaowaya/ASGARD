# F0.11 孤立森林（Isolation Forest）

## 算法概述

孤立森林是一种无监督异常检测算法，基于异常点容易被孤立（路径短）的特性。在电池管理系统中，孤立森林主要用于：

- **内短路检测**：早期识别内短路电芯
- **异常电芯识别**：发现偏离正常模式的电芯
- **故障预警**：实时监控异常行为

## 核心原理

**核心思想**：异常点容易被孤立（路径短）

**步骤**：
1. 随机选择特征和分割点
2. 递归划分特征空间
3. 计算路径长度（path length）
4. 异常分数 = 2^(-E(path)/c(n))

**其中**：
- E(path): 平均路径长度
- c(n): 标准化因子

**优势**：
- 无需标注
- 对高维数据效果好
- 计算效率高

## 代码示例

```python
from sklearn.ensemble import IsolationForest
import numpy as np

def internal_short_circuit_detection(voltage_matrix):
    """
    基于孤立森林的内短路检测

    输入：
        voltage_matrix: (n_cells, n_timesteps)
        每行是一个电芯的电压时序

    输出：
        anomaly_scores: 异常分数
        isc_cells: 内短路电芯索引
    """
    # 特征提取：统计特征
    features = np.column_stack([
        voltage_matrix.mean(axis=1),      # 平均电压
        voltage_matrix.std(axis=1),       # 电压波动
        voltage_matrix.min(axis=1),       # 最低电压
        voltage_matrix.ptp(axis=1)        # 电压极差
    ])

    # 孤立森林
    model = IsolationForest(
        contamination=0.05,  # 预期异常比例
        random_state=42,
        n_jobs=-1
    )
    model.fit(features)

    # 异常分数（-1为异常，1为正常）
    labels = model.predict(features)
    anomaly_scores = model.score_samples(features)  # 越低越异常

    # 提取异常电芯
    isc_cells = np.where(labels == -1)[0]

    return anomaly_scores, isc_cells
```
