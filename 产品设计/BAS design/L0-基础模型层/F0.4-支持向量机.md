# F0.4 支持向量机（SVM/SVR）

## 算法概述

支持向量机（SVM）是一种强大的监督学习算法，通过寻找最优超平面来实现分类和回归。在电池管理系统中，SVM主要用于：

- **小样本分类**：电池故障分类、状态识别
- **回归预测**：SOH估计、RUL预测
- **异常检测**：识别偏离正常模式的样本

## 核心原理

### SVM分类

**目标**：找到最大间隔超平面

```
min_{w,b} ½||w||²
s.t. y_i(w·x_i + b) ≥ 1

其中：
- w: 法向量
- b: 偏置
- y_i: 标签（±1）
```

### SVR回归

**目标**：找到ε-不敏感损失函数下的最优超平面

```
min_{w,b} ½||w||² + CΣξ_i
s.t. |y_i - (w·x_i + b)| ≤ ε + ξ_i

其中：
- ε: 不敏感区
- C: 惩罚系数
- ξ_i: 松弛变量
```

### 核技巧

**处理非线性问题**：
```
K(x_i, x_j) = φ(x_i)·φ(x_j)

常用核函数：
- RBF核：exp(-γ||x_i - x_j||²)
- 多项式核：(γx_i·x_j + r)^d
- Sigmoid核：tanh(γx_i·x_j + r)
```

## 代码示例

```python
from sklearn.svm import SVC, SVR
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import accuracy_score, mean_squared_error
import numpy as np

# SVM分类示例
def battery_fault_classification(features, labels):
    """
    基于SVM的电池故障分类

    参数:
        features: 特征矩阵 (n_samples, n_features)
        labels: 故障标签 (n_samples,)

    返回:
        model: 训练好的SVM模型
        accuracy: 分类准确率
    """
    # 标准化
    scaler = StandardScaler()
    X_std = scaler.fit_transform(features)

    # 训练SVM
    model = SVC(
        kernel='rbf',      # RBF核
        C=1.0,            # 惩罚系数
        gamma='scale',    # 核系数
        probability=True  # 输出概率
    )
    model.fit(X_std, labels)

    # 预测
    labels_pred = model.predict(X_std)
    accuracy = accuracy_score(labels, labels_pred)

    return model, accuracy

# SVR回归示例
def soh_svr_regression(features, soh_true):
    """
    基于SVR的SOH估计

    参数:
        features: 特征矩阵
        soh_true: 真实SOH值

    返回:
        soh_pred: 预测SOH
        rmse: 均方根误差
    """
    # 标准化
    scaler = StandardScaler()
    X_std = scaler.fit_transform(features)

    # 训练SVR
    model = SVR(
        kernel='rbf',
        C=1.0,
        gamma='scale',
        epsilon=0.1
    )
    model.fit(X_std, soh_true)

    # 预测
    soh_pred = model.predict(X_std)
    rmse = np.sqrt(mean_squared_error(soh_true, soh_pred))

    return soh_pred, rmse
```

## 应用场景

- **故障分类**：正常/析锂/内短路/热失控
- **SOH估计**：小样本高精度估计
- **状态识别**：充电/放电/静置
