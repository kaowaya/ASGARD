# F0.29 Seq2Seq序列到序列

## 算法概述

Seq2Seq是一种Encoder-Decoder架构，用于将一个序列映射到另一个序列。在电池管理系统中，Seq2Seq可用于：

- **序列转换**：电压曲线→电流曲线
- **多步预测**：历史→未来
- **数据修复**：异常值修正

## 架构

```
Encoder: 输入序列 → 隐含状态
Decoder: 隐含状态 → 输出序列
```

## 代码示例

```python
import torch
import torch.nn as nn

class Seq2Seq(nn.Module):
    """
    Seq2Seq模型
    """
    def __init__(self, input_dim, hidden_dim, output_dim, n_layers=2):
        super().__init__()

        # Encoder
        self.encoder = nn.LSTM(input_dim, hidden_dim, n_layers, batch_first=True)

        # Decoder
        self.decoder = nn.LSTM(output_dim, hidden_dim, n_layers, batch_first=True)

        # 输出层
        self.fc = nn.Linear(hidden_dim, output_dim)

    def forward(self, src, trg, teacher_forcing_ratio=0.5):
        """
        src: 源序列 (batch, src_len, input_dim)
        trg: 目标序列 (batch, trg_len, output_dim)
        """
        batch_size = src.size(0)
        trg_len = trg.size(1)

        # Encoder
        _, (hidden, cell) = self.encoder(src)

        # Decoder
        outputs = []
        input = trg[:, 0:1, :]  # 第一个时间步

        for t in range(1, trg_len):
            output, (hidden, cell) = self.decoder(input, (hidden, cell))
            prediction = self.fc(output)

        outputs = torch.cat(outputs, dim=1)

        return outputs
```
