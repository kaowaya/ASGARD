# F0.39 遗传算法（Genetic Algorithm）

## 算法概述

遗传算法模拟生物进化过程，通过选择、交叉、变异寻找全局最优解。在电池管理系统中，GA主要用于：

- **参数辨识**：电池模型参数优化
- **模型训练**：神经网络超参数
- **多目标优化**：帕累托前沿

## 算法步骤

```
1. 初始化种群：随机生成N个解
2. 评估适应度：计算每个解的目标函数值
3. 选择：根据适应度选择父代（轮盘赌、锦标赛）
4. 交叉：父代配对生成子代（单点、多点交叉）
5. 变异：随机扰动基因（变异率）
6. 替换：子代替换父代
7. 重复2-6直到收敛
```

## 代码示例

```python
import numpy as np

def genetic_algorithm(objective_func, bounds, n_generations=100, pop_size=50):
    """
    遗传算法优化

    参数：
        objective_func: 目标函数（最小化）
        bounds: 参数边界 [(min_1, max_1), ..., (min_n, max_n)]
        n_generations: 迭代次数
        pop_size: 种群大小
    """
    n_vars = len(bounds)

    # 1. 初始化种群
    population = np.random.uniform(
        low=[b[0] for b in bounds],
        high=[b[1] for b in bounds],
        size=(pop_size, n_vars)
    )

    # 进化循环
    for gen in range(n_generations):
        # 2. 评估适应度
        fitness = np.array([objective_func(ind) for ind in population])

        # 3. 选择（锦标赛）
        selected = []
        for _ in range(pop_size):
            candidates = np.random.choice(pop_size, size=3, replace=False)
            winner = candidates[np.argmin(fitness[candidates])]
            selected.append(population[winner])
        selected = np.array(selected)

        # 4. 交叉（单点交叉）
        offspring = []
        for i in range(0, pop_size, 2):
            parent1, parent2 = selected[i], selected[i+1]
            crossover_point = np.random.randint(1, n_vars)
            child1 = np.concatenate([parent1[:crossover_point], parent2[crossover_point:]])
            child2 = np.concatenate([parent2[:crossover_point], parent1[crossover_point:]])
            offspring.extend([child1, child2])

        # 5. 变异
        offspring = np.array(offspring)
        mutation_rate = 0.1
        for i in range(pop_size):
            if np.random.rand() < mutation_rate:
                var_idx = np.random.randint(0, n_vars)
                offspring[i, var_idx] = np.random.uniform(
                    bounds[var_idx][0],
                    bounds[var_idx][1]
                )

        # 6. 替换
        population = offspring

        # 打印进度
        if gen % 10 == 0:
            best_fitness = fitness.min()
            print(f"Generation {gen}, Best Fitness: {best_fitness:.6f}")

    # 返回最优解
    final_fitness = np.array([objective_func(ind) for ind in population])
    best_idx = np.argmin(final_fitness)

    return population[best_idx], final_fitness[best_idx]
```
