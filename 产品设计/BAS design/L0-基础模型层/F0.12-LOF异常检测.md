# F0.12 LOF异常检测（Local Outlier Factor）

## 算法概述

LOF是一种基于局部密度的异常检测算法，能够发现局部偏离模式的异常点。在电池管理系统中，LOF特别适合：

- **局部异常检测**：识别局部区域的异常电芯
- **电压突变识别**：检测电压曲线突变
- **多模态数据**：处理多模态分布

## 核心原理

**核心思想**：基于局部密度的离群点检测

**步骤**：
1. 计算每个点的k距离（k-distance）
2. 计算局部可达密度（LRD）
3. 计算LOF分数：
   ```
   LOF(p) = (Σ_{q∈N_k(p)} LRD(q) / LRD(p)) / k
   ```

**解释**：
- LOF ≈ 1: 正常点（密度与邻居相似）
- LOF >> 1: 异常点（密度低于邻居）
- LOF << 1: 高密度点（聚类中心）

**优势**：
- 可发现局部异常
- 自适应阈值
- 适合多模态数据

## 代码示例

```python
from sklearn.neighbors import LocalOutlierFactor
import numpy as np

def voltage_anomaly_detection_lof(voltage_matrix, n_neighbors=20):
    """
    基于LOF的电压异常检测

    输入：
        voltage_matrix: (n_cells, n_timesteps)

    输出：
        lof_scores: LOF分数
        anomaly_indices: 异常点索引
    """
    # 特征提取
    features = np.column_stack([
        voltage_matrix.mean(axis=1),
        voltage_matrix.std(axis=1),
        voltage_matrix.min(axis=1),
        voltage_matrix.max(axis=1)
    ])

    # LOF检测
    model = LocalOutlierFactor(
        n_neighbors=n_neighbors,
        contamination=0.05,
        novelty=True  # 允许predict
    )
    model.fit(features)

    # LOF分数
    lof_scores = -model.negative_outlier_factor_  # 转为正数

    # 预测（-1为异常，1为正常）
    labels = model.predict(features)
    anomaly_indices = np.where(labels == -1)[0]

    return lof_scores, anomaly_indices
```
