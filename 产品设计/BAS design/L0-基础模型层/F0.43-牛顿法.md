# F0.43 牛顿法（Newton Method）

## 算法概述

牛顿法利用二阶导数（Hessian矩阵）进行优化，具有二次收敛速度。在电池管理系统中，牛顿法主要用于：

- **无约束优化**：光滑目标函数
- **参数辨识**：快速收敛
- **在线优化**：实时优化

## 更新公式

```
x_{k+1} = x_k - [Hf(x_k)]^{-1} · ∇f(x_k)

其中：
- ∇f: 梯度（一阶导数）
- Hf: Hessian矩阵（二阶导数）

优势：收敛速度快（二次收敛）
劣势：需要计算Hessian矩阵（计算量大）
```

## 代码示例

```python
import numpy as np

def newton_method(objective_func, gradient_func, hessian_func, x0, max_iter=100, tol=1e-6):
    """
    牛顿法优化

    参数：
        objective_func: 目标函数
        gradient_func: 梯度函数
        hessian_func: Hessian矩阵函数
        x0: 初始点
        max_iter: 最大迭代次数
        tol: 收敛容忍度

    返回：
        x: 最优解
        f_val: 最优目标函数值
    """
    x = x0.copy()

    for iter in range(max_iter):
        # 计算梯度和Hessian
        grad = gradient_func(x)
        hess = hessian_func(x)

        # 检查Hessian是否正定
        try:
            # 牛顿步
            delta = np.linalg.solve(hess, grad)
        except np.linalg.LinAlgError:
            print("Hessian矩阵非正定，添加正则化项")
            hess_reg = hess + 1e-6 * np.eye(len(x))
            delta = np.linalg.solve(hess_reg, grad)

        # 更新
        x_new = x - delta

        # 检查收敛
        if np.linalg.norm(delta) < tol:
            print(f"收敛于迭代 {iter}")
            break

        x = x_new

        # 打印进度
        if iter % 10 == 0:
            f_val = objective_func(x)
            print(f"Iteration {iter}, f(x): {f_val:.6f}")

    f_val = objective_func(x)

    return x, f_val

# 应用：二次函数优化
def quadratic_objective(x):
    """二次目标函数"""
    Q = np.array([[2, 1], [1, 2]])
    return 0.5 * x.T @ Q @ x

def quadratic_gradient(x):
    """梯度"""
    Q = np.array([[2, 1], [1, 2]])
    return Q @ x

def quadratic_hessian(x):
    """Hessian矩阵"""
    return np.array([[2, 1], [1, 2]])

# 优化
x0 = np.array([1.0, 1.0])
x_opt, f_opt = newton_method(
    quadratic_objective,
    quadratic_gradient,
    quadratic_hessian,
    x0
)

print(f"最优解: {x_opt}")
print(f"最优值: {f_opt}")
```

## 改进：拟牛顿法

**BFGS**：
```
不需要显式计算Hessian，通过梯度近似
B_{k+1} = B_k + ...（BFGS更新公式）
```

**L-BFGS**：
```
限制内存的BFGS，适合大规模问题
```

## 代码示例：BFGS

```python
from scipy.optimize import minimize

def bfgs_optimization(objective_func, gradient_func, x0):
    """
    BFGS优化（拟牛顿法）
    """
    result = minimize(
        objective_func,
        x0,
        method='BFGS',
        jac=gradient_func,
        options={'disp': True}
    )

    if result.success:
        return result.x, result.fun
    else:
        return None, None

# 应用
x0 = np.array([1.0, 1.0])
x_opt, f_opt = bfgs_optimization(
    quadratic_objective,
    quadratic_gradient,
    x0
)
```
