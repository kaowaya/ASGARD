## 算法原理

### OCV-SOC查表

```cpp
// LFP电池OCV-SOC映射表
const struct { float soc; float ocv; } ocv_soc_table[] = {
    {0.0f, 2.80f}, {0.1f, 3.20f}, {0.2f, 3.25f},
    {0.3f, 3.27f}, {0.4f, 3.28f}, {0.5f, 3.29f},
    {0.6f, 3.30f}, {0.7f, 3.31f}, {0.8f, 3.32f},
    {0.9f, 3.35f}, {1.0f, 3.60f}
};

float lookup_soc_by_ocv(float ocv) {
    // 线性插值
    for (int i = 0; i < sizeof(ocv_soc_table)/sizeof(ocv_soc_table[0]) - 1; i++) {
        if (ocv >= ocv_soc_table[i].ocv && ocv <= ocv_soc_table[i+1].ocv) {
            float ratio = (ocv - ocv_soc_table[i].ocv) /
                         (ocv_soc_table[i+1].ocv - ocv_soc_table[i].ocv);
            return ocv_soc_table[i].soc +
                   ratio * (ocv_soc_table[i+1].soc - ocv_soc_table[i].soc);
        }
    }
    return -1.0f;  // 超出范围
}
```

### 修正条件

- 静置时间 > 2小时（充分弛豫）
- 电流 < 0.01C（接近开路）
