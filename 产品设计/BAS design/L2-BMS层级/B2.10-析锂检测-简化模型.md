## 算法原理

### 负极电位估计

$$
\Phi_{anode} = OCV_{anode}(SOC_{anode}) - I \cdot R_{anode}
$$

### 析锂判据

$$
\Phi_{anode} < 0 \text{ V (vs. Li/Li}^+)
$$

### 简化实现（无需完整电化学模型）

```python
def detect_lithium_plating_simple(voltage, current, ocv_anode_table):
    """
    简化析锂检测
    Args:
        voltage: 端电压
        current: 充电电流（正）
        ocv_anode_table: 负极OCV-SOC表
    Returns:
        is_plating: 是否析锂
        risk_level: 风险等级（0-1）
    """
    # 估算负极SOC（简化为正极SOC - 偏移）
    soc_anode = estimate_soc_from_voltage(voltage) - 0.1  # N/P比假设

    # 负极OCV
    ocv_anode = np.interp(soc_anode,
                          ocv_anode_table['soc'],
                          ocv_anode_table['ocv'])

    # 过电位
    eta = current * 0.01  # 简化内阻

    # 负极电位
    phi_anode = ocv_anode - eta

    # 析锂判据
    if phi_anode < 0:
        risk_level = abs(phi_anode) / 0.05  # 归一化
        return True, min(risk_level, 1.0)
    else:
        return False, 0.0
```
