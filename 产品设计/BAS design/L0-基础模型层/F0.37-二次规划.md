# F0.37 二次规划（Quadratic Programming）

## 算法概述

二次规划是目标函数为二次、约束为线性的优化方法。在电池管理系统中，QP主要用于：

- **MPC控制**：模型预测控制
- **最优控制**：轨迹优化
- **投资组合**：风险管理

## 标准形式

```
min ½x^T Q x + c^T x
s.t. Ax ≤ b
     x ≥ 0
```

## 应用示例：模型预测控制（MPC）

```
min Σ||T(k) - T_ref||² + λ·P_cooling(k)²
s.t. T_min ≤ T(k) ≤ T_max
     P_cooling_min ≤ P_cooling(k) ≤ P_cooling_max
```

## 代码示例

```python
from scipy.optimize import minimize
import numpy as np

def mpc_temperature_control(T_current, T_ref, horizon=10):
    """
    MPC温度控制

    参数:
        T_current: 当前温度
        T_ref: 目标温度
        horizon: 预测时域

    返回:
        cooling_power: 最优冷却功率序列
    """
    # 决策变量：[P(0), P(1), ..., P(horizon-1)]
    n_vars = horizon

    # 目标函数：min Σ(T(k) - T_ref)² + λ·P(k)²
    def objective(x):
        # 简化的温度预测模型
        T_pred = T_current
        cost = 0
        lambda_cooling = 0.1

        for k in range(horizon):
            # 温度动态：T(k+1) = T(k) - 0.1*P(k) + 0.5*I(k)
            T_pred = T_pred - 0.1 * x[k] + 0.5 * 100  # 假设电流100A
            cost += (T_pred - T_ref)**2 + lambda_cooling * x[k]**2

        return cost

    # 约束
    bounds = [(0, 1000) for _ in range(n_vars)]  # 冷却功率限制0-1000W

    # 求解
    x0 = np.zeros(n_vars)  # 初始猜测
    result = minimize(objective, x0, bounds=bounds, method='SLSQP')

    if result.success:
        cooling_power = result.x
        print(f"最优冷却功率序列: {cooling_power}")
        return cooling_power
    else:
        print("优化失败")
        return None

# 应用示例
if __name__ == "__main__":
    T_current = 35  # 当前温度35°C
    T_ref = 25      # 目标温度25°C

    cooling_power = mpc_temperature_control(T_current, T_ref)
```
