# F0.7 LightGBM

## 算法概述

LightGBM（Light Gradient Boosting Machine）是微软开发的轻量级梯度提升框架，基于直方图的决策树算法。相比XGBoost，LightGBM训练速度更快、内存占用更低。

## 核心原理

**核心创新**：基于直方图的决策树

**vs XGBoost**：
- 训练速度快（直方图加速）
- 内存占用低
- 精度相当
- 适合大数据集

**关键特性**：
- GOSS（Gradient-based One-Side Sampling）：梯度采样
- EFB（Exclusive Feature Bundling）：特征捆绑
- Leaf-wise生长策略：更深的树

## 代码示例

```python
import lightgbm as lgb
from sklearn.metrics import mean_squared_error
import numpy as np

def soh_lightgbm(X_train, y_train, X_test, y_test):
    """
    基于LightGBM的SOH估计
    """
    # Dataset格式
    train_data = lgb.Dataset(X_train, label=y_train)
    test_data = lgb.Dataset(X_test, label=y_test, reference=train_data)

    # 参数
    params = {
        'boosting_type': 'gbdt',
        'objective': 'regression',
        'metric': 'rmse',
        'num_leaves': 31,
        'learning_rate': 0.05,
        'feature_fraction': 0.9,
        'bagging_fraction': 0.8,
        'bagging_freq': 5,
        'verbose': 0
    }

    # 训练
    gbm = lgb.train(
        params,
        train_data,
        num_boost_round=100,
        valid_sets=[test_data],
        early_stopping_rounds=10,
        verbose_eval=False
    )

    # 预测
    soh_pred = gbm.predict(X_test, num_iteration=gbm.best_iteration)

    # 评估
    rmse = np.sqrt(mean_squared_error(y_test, soh_pred))

    return gbm, soh_pred, rmse
```
