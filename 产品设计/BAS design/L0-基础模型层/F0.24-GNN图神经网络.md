# F0.24 GNN图神经网络

## 算法概述

图神经网络（GNN）是专门处理图结构数据的深度学习架构。在电池管理系统中，GNN主要用于：

- **电池包一致性分析**：电芯间关系建模
- **热失控传播预测**：预测热扩散路径
- **故障定位**：识别故障电芯
- **均衡策略优化**：主动均衡决策

## 图卷积公式

```
H^(l+1) = σ(Ã^(l) · H^(l) · W^(l))

其中：
- Ã^(l): 归一化邻接矩阵
- H^(l): 第l层节点特征
- W^(l): 可学习权重
- σ: 激活函数
```

## 代码示例

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GCNConv

class BatteryGNN(nn.Module):
    """
    基于GNN的电池包一致性分析

    输入：
        x: 节点特征 (num_nodes, node_features)
        edge_index: 边索引 (2, num_edges)

    输出：
        node_embeddings: 节点嵌入
    """
    def __init__(self, node_features, hidden_dim, output_dim):
        super().__init__()

        # 图卷积层
        self.conv1 = GCNConv(node_features, hidden_dim)
        self.conv2 = GCNConv(hidden_dim, hidden_dim)
        self.conv3 = GCNConv(hidden_dim, output_dim)

    def forward(self, x, edge_index):
        # GCN层
        x = self.conv1(x, edge_index)
        x = F.relu(x)
        x = F.dropout(x, p=0.5, training=self.training)

        x = self.conv2(x, edge_index)
        x = F.relu(x)

        x = self.conv3(x, edge_index)

        return x

# 应用：热失控传播预测
class ThermalPropagationGNN(nn.Module):
    """
    热失控传播预测
    """
    def __init__(self, node_features, hidden_dim):
        super().__init__()

        self.conv1 = GCNConv(node_features, hidden_dim)
        self.conv2 = GCNConv(hidden_dim, hidden_dim)

        # 预测层
        self.fc = nn.Linear(hidden_dim, 1)  # 输出热失控概率

    def forward(self, x, edge_index):
        x = self.conv1(x, edge_index)
        x = F.relu(x)
        x = self.conv2(x, edge_index)
        x = F.relu(x)

        # 热失控概率
        thermal_prob = torch.sigmoid(self.fc(x))

        return thermal_prob
```
