## 算法原理

### 双自适应机制

```
1. Sage-Husa自适应噪声估计
   Q_k = (1-d_k)·Q_{k-1} + d_k·K_k·ε_k·ε_kᵀ·K_kᵀ
   R_k = (1-d_k)·R_{k-1} + d_k·ε_k·ε_kᵀ

   其中遗忘因子:
   d_k = (1-b) / (1-b^(k+1)), b=0.97

2. 遗忘因子自适应
   检测工况突变 → 减小b，加速收敛
```

## 代码示例

```python
class AEKF_SOC(EKF_SOC):
    def __init__(self, *args, b=0.97, **kwargs):
        super().__init__(*args, **kwargs)
        self.b = b  # Sage-Husa遗忘因子
        self.k = 0  # 时间步计数

    def update(self, V_meas, I):
        # 观测预测
        V_pred = self.ocv_func(self.x[0]) - I*self.R0 - self.x[1] - self.x[2]

        # 新息
        epsilon = V_meas - V_pred

        # 观测矩阵
        H = np.array([[0.4, -1, -1]])

        # Sage-Husa自适应噪声估计
        d_k = (1 - self.b) / (1 - self.b**(self.k + 1))

        # 更新R
        self.R = (1 - d_k) * self.R + d_k * epsilon**2

        # 卡尔曼增益
        S = H @ self.P @ H.T + self.R
        K = self.P @ H.T / S

        # 状态更新
        self.x = self.x + K.flatten() * epsilon

        # 更新Q
        self.Q = (1 - d_k) * self.Q + d_k * np.outer(K, K) * epsilon**2

        # 协方差更新
        self.P = (np.eye(3) - np.outer(K, H)) @ self.P

        self.k += 1
```
