## 算法原理

### 约束条件

```
电压约束:
  V_min ≤ OCV(SOC) - I·R_0 - V_p1 - V_p2 ≤ V_max

SOC约束:
  SOC_min ≤ SOC - I·Δt/C_n ≤ SOC_max

电流约束:
  I_min ≤ I ≤ I_max

温度约束:
  T_min ≤ T_pred(I) ≤ T_max
```

### SOP计算

```python
def calculate_sop(soc, voltage, temp, R0, constraints):
    """
    计算持续功率能力（10秒、30秒、瞬时）
    """
    # 10秒持续放电功率
    I_10s = min(
        (voltage - constraints['V_min']) / R0,  # 电压限制
        (soc - constraints['SOC_min']) * 360 * 10,  # SOC限制
        constraints['I_max']  # 电流限制
    )

    P_10s = I_10s * voltage

    # 30秒、瞬时同理...

    return {
        'P_10s': P_10s,
        'P_30s': P_30s,
        'P_peak': P_peak
    }
```
