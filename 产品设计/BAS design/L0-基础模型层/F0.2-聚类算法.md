# F0.2 聚类算法（K-Means、DBSCAN、GMM）

## 算法概述

聚类算法将相似的数据样本归为一组，无需标签即可发现数据的内在结构。在电池管理系统中，聚类算法主要用于：

- **电芯分级**：根据OCV、DCIR、K值等参数将电芯分为A/B/C级
- **异常检测**：识别偏离正常模式的异常电芯
- **用户分群**：基于充电行为对用户进行分群
- **工况识别**：自动识别充电、放电、静置等工况

## 聚类算法选择指南

| 场景 | 推荐算法 | 理由 |
|------|---------|------|
| **已知类别数** | K-Means | 简单高效，适合球状簇 |
| **未知类别数、含噪声** | DBSCAN | 自动发现类别，识别异常点 |
| **软聚类（概率输出）** | GMM | 提供后验概率，适合重叠簇 |

## 核心原理

### K-Means

**目标**：最小化簇内平方误差和

**步骤**：
1. 初始化：随机选择k个中心点
2. 分配：将每个样本分配到最近的中心
3. 更新：重新计算每个簇的中心
4. 重复2-3直到收敛

**目标函数**：
```
J = Σ_{i=1}^k Σ_{x∈C_i} ||x - μ_i||²

其中：
- C_i: 第i个簇
- μ_i: 第i个簇的中心
- ||·||: 欧氏距离
```

### DBSCAN

**核心思想**：基于密度的聚类，自动发现类别

**参数**：
- ε: 邻域半径
- MinPts: 最小样本数

**步骤**：
1. 对每个样本，计算ε邻域内的样本数
2. 如果邻域内样本数 ≥ MinPts，创建新簇
3. 递归扩展簇（密度可达）
4. 无法归入任何簇的样本为噪声

**优势**：
- 自动发现类别数量
- 能识别任意形状的簇
- 自动检测噪声点

### GMM（高斯混合模型）

**假设**：数据来自k个高斯分布的混合

**模型**：
```
p(x) = Σ_{i=1}^k π_i · N(x|μ_i, Σ_i)

其中：
- π_i: 第i个高斯分布的混合系数（Σπ_i = 1）
- μ_i: 第i个高斯分布的均值
- Σ_i: 第i个高斯分布的协方差矩阵
```

**学习算法**：EM算法
- E步：计算后验概率 `γ(z_{ni}) = p(z_{ni}|x_n)`
- M步：最大化期望对数似然，更新参数

## 代码示例

```python
from sklearn.cluster import KMeans, DBSCAN
from sklearn.mixture import GaussianMixture
import numpy as np

def battery_cell_grading(features, method='KMEANS', **kwargs):
    """
    电池电芯分级聚类

    参数:
        features: 特征矩阵 (n_cells, n_features)
        method: 'KMEANS', 'DBSCAN', or 'GMM'
        **kwargs: 算法特定参数

    返回:
        labels: 聚类标签 (n_cells,)
        centers: 簇中心
        metrics: 评估指标
    """

    if method == 'KMEANS':
        # K-Means: 适合已知等级数量（例如：A/B/C三级）
        n_clusters = kwargs.get('n_clusters', 3)
        model = KMeans(
            n_clusters=n_clusters,
            init='k-means++',
            n_init=10,
            max_iter=300,
            random_state=42
        )
        labels = model.fit_predict(features)
        centers = model.cluster_centers_

        # 评估指标
        inertia = model.inertia_  # 簇内平方误差和
        metrics = {'inertia': inertia}

    elif method == 'DBSCAN':
        # DBSCAN: 适合自动发现异常电芯
        eps = kwargs.get('eps', 0.5)
        min_samples = kwargs.get('min_samples', 5)

        model = DBSCAN(
            eps=eps,
            min_samples=min_samples,
            metric='euclidean'
        )
        labels = model.fit_predict(features)

        # DBSCAN没有显式的簇中心
        centers = None

        # 识别噪声点（标签为-1）
        n_noise = list(labels).count(-1)
        metrics = {
            'n_clusters': len(set(labels)) - (1 if -1 in labels else 0),
            'n_noise': n_noise
        }

    elif method == 'GMM':
        # GMM: 提供概率输出
        n_components = kwargs.get('n_components', 3)
        model = GaussianMixture(
            n_components=n_components,
            covariance_type='full',
            max_iter=100,
            random_state=42
        )
        model.fit(features)
        labels = model.predict(features)
        centers = model.means_

        # 后验概率（每个样本属于每个簇的概率）
        probs = model.predict_proba(features)
        metrics = {'probabilities': probs}

    return labels, centers, metrics

# 应用场景：电芯OCV内阻分级
def cell_ocv_dcir_grading(ocv, dcir):
    """
    基于OCV和DCIR的电芯分级

    输入:
        ocv: 开路电压 (n_cells,)
        dcir: 直流内阻 (n_cells,)

    输出:
        grade: A/B/C等级
    """
    # 特征构造
    features = np.column_stack([ocv, dcir])

    # 标准化
    features = (features - features.mean(axis=0)) / features.std(axis=0)

    # K-Means聚类（3级）
    labels, centers, _ = battery_cell_grading(
        features,
        method='KMEANS',
        n_clusters=3
    )

    # 根据中心点排序确定等级
    center_order = np.argsort(centers[:, 0])  # 按OCV排序

    # 映射到A/B/C等级
    grade_mapping = {}
    for i, center_idx in enumerate(center_order):
        grade_mapping[center_idx] = ['C', 'B', 'A'][i]  # 低OCV为C级

    grades = [grade_mapping[label] for label in labels]

    return grades

# 应用场景：异常电芯检测
def anomaly_detection(features):
    """
    DBSCAN检测异常电芯
    """
    # DBSCAN聚类
    labels, _, metrics = battery_cell_grading(
        features,
        method='DBSCAN',
        eps=0.5,
        min_samples=5
    )

    # 提取异常点（噪声）
    anomaly_mask = labels == -1
    n_anomalies = anomaly_mask.sum()

    print(f"发现 {n_anomalies} 个异常电芯（共 {len(features)} 个）")

    return anomaly_mask
```

## 应用场景

### 场景1：生产质量控制 - 电芯分级
```
输入：电芯OCV、DCIR、K值
处理：K-Means聚类（k=3）
输出：A级（优秀）、B级（良好）、C级（不合格）
价值：自动化分级，提高一致性
```

### 场景2：运维异常检测
```
输入：储能电站电芯电压、温度、内阻
处理：DBSCAN聚类
输出：正常簇 + 异常点
价值：早期发现潜在故障电芯
```

- 磷酸铁锂（LFP）

## 部署建议

### 云端部署
- **适用场景**：生产质检、大规模数据分析
- **推荐配置**：8GB+ RAM
- **批量处理**：支持批量处理数万电芯数据

### 边缘端部署
- **适用场景**：本地异常检测
- **推荐算法**：K-Means（推理速度快）
- **优化方案**：
  - 预训练K-Means模型
  - 使用MiniBatchKMeans降低内存占用
  - 缓存簇中心，仅计算距离

## 性能优化

1. **数据预处理**：
   - 标准化：消除量纲影响
   - 降维：PCA预处理加速聚类

2. **参数调优**：
   - K-Means：使用肘部法或轮廓系数选择k
   - DBSCAN：使用k-距离图选择eps
   - GMM：使用BIC或AIC选择分量数

3. **加速技巧**：
   - K-Means：使用MiniBatchKMeans处理大数据
   - 初始化：使用k-means++加速收敛
   - 并行化：设置n_jobs参数

## 评估指标

- **轮廓系数**（Silhouette Score）：-1到1，越接近1越好
- **Calinski-Harabasz指数**：越大越好
- **Davies-Bouldin指数**：越小越好
