# F0.6 XGBoost

## 算法概述

XGBoost（eXtreme Gradient Boosting）是一种优化的分布式梯度提升库，以其出色的性能和速度著称。在电池管理系统中，XGBoost是SOH预测、RUL估计等任务的SOTA模型。

## 核心原理

**核心思想**：Boosting + 梯度下降

**目标函数**：
```
Obj = ΣL(y_i, ŷ_i) + ΣΩ(f_k)

其中：
- L: 损失函数（MSE、LogLoss等）
- Ω: 正则化项（防止过拟合）
```

**优化**：贪心算法，每次添加一棵树来拟合残差

**优势**：
- 精度高（SOTA表现）
- 自动处理缺失值
- 支持并行计算
- 可解释性强（SHAP值）

## 代码示例

```python
import xgboost as xgb
from sklearn.metrics import mean_squared_error
import numpy as np

def soh_xgboost(X_train, y_train, X_test, y_test):
    """
    基于XGBoost的SOH估计
    """
    # DMatrix格式
    dtrain = xgb.DMatrix(X_train, label=y_train)
    dtest = xgb.DMatrix(X_test, label=y_test)

    # 参数
    params = {
        'max_depth': 6,
        'eta': 0.1,
        'objective': 'reg:squarederror',
        'eval_metric': 'rmse',
        'subsample': 0.8,
        'colsample_bytree': 0.8
    }

    # 训练
    evals_result = {}
    bst = xgb.train(
        params,
        dtrain,
        num_boost_round=100,
        evals=[(dtrain, 'train'), (dtest, 'test')],
        evals_result=evals_result,
        early_stopping_rounds=10,
        verbose_eval=False
    )

    # 预测
    soh_pred = bst.predict(dtest)

    # 特征重要性
    importance = bst.get_score(importance_type='gain')

    # 评估
    rmse = np.sqrt(mean_squared_error(y_test, soh_pred))

    return bst, soh_pred, importance, rmse

# SHAP值解释
import shap

def explain_with_shap(model, X, feature_names):
    """
    使用SHAP值解释模型
    """
    explainer = shap.TreeExplainer(model)
    shap_values = explainer.shap_values(X)

    # 可视化
    shap.summary_plot(shap_values, X, feature_names=feature_names)

    return shap_values
```
