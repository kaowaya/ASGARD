# F0.36 线性规划（Linear Programming）

## 算法概述

线性规划是一种优化方法，用于在线性约束条件下优化线性目标函数。在电池管理系统中，线性规划主要用于：

- **能量管理**：储能电站功率分配
- **资源分配**：充放电资源优化
- **调度优化**：多电池系统调度

## 标准形式

```
min c^T x
s.t. Ax ≤ b
     x ≥ 0
```

## 应用示例：储能电站功率分配

```
min Σcost_i · P_i
s.t. ΣP_i = P_load
     P_min ≤ P_i ≤ P_max
     SOC_min ≤ SOC ≤ SOC_max
```

## 代码示例

```python
from scipy.optimize import linprog
import numpy as np

def energy_management_system(costs, power_demand, power_limits):
    """
    储能电站能量管理

    参数:
        costs: 各单元发电成本
        power_demand: 功率需求
        power_limits: 功率限制 [(min_1, max_1), ..., (min_n, max_n)]

    返回:
        optimal_power: 最优功率分配
    """
    n_units = len(costs)

    # 目标函数系数
    c = costs

    # 约束矩阵
    A_eq = [np.ones(n_units)]  # 功率平衡约束
    b_eq = [power_demand]

    # 边界
    bounds = power_limits

    # 求解
    result = linprog(c, A_eq=A_eq, b_eq=b_eq, bounds=bounds, method='highs')

    if result.success:
        optimal_power = result.x
        total_cost = result.fun
        print(f"最优功率分配: {optimal_power}")
        print(f"总成本: {total_cost:.2f}")
        return optimal_power, total_cost
    else:
        print("优化失败")
        return None, None

# 应用示例
if __name__ == "__main__":
    # 3个储能单元
    costs = [10, 15, 12]  # 成本
    power_demand = 1000   # 需求1000kW
    power_limits = [(0, 500), (0, 600), (0, 400)]  # 功率限制

    optimal_power, total_cost = energy_management_system(
        costs, power_demand, power_limits
    )
```
