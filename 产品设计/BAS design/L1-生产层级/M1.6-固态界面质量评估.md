## 算法原理

### 1. 固态电池EIS特征

```
固态电池特有现象: 界面接触不良 → 高频半圆

等效电路: L-R_s-(R_int|C_int)-(R_ct|C_dl)-Z_W

  其中:
    R_int: 界面接触电阻（固态特有）
    C_int: 界面电容
    R_ct: 电荷转移电阻
    C_dl: 双电层电容
```

### 2. 界面质量指标

```
界面因子 IF = R_int / (R_int + R_ct)

  - IF < 0.2: 优秀
  - 0.2 ≤ IF < 0.4: 良好
  - IF ≥ 0.4: 需改进
```

---

## 代码示例

```python
from impedance.models.circuits import CustomCircuit

# 固态电池专用电路
circuit = 'L-R0-p(R1,C1)-p(R2,C2)-W1'
model = CustomCircuit(circuit,
                      initial_guess=[1e-8, 0.01, 1000, 0.1, 0.05, 500, 0.1])

# 拟合
model.fit(frequencies, Z_measured)

# 计算界面因子
R_int = model.parameters_[1]
R_ct = model.parameters_[3]
IF = R_int / (R_int + R_ct)

if IF < 0.2:
    quality = "优秀"
elif IF < 0.4:
    quality = "良好"
else:
    quality = "需改进"

print(f"界面因子: {IF:.3f} ({quality})")
```
