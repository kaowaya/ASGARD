# F0.17 梯度提升机（Gradient Boosting Machine）

## 算法概述

GBM是一种提升算法，通过迭代添加弱学习器来拟合前一个模型的残差。在电池管理系统中，GBM可用于：

- **SOH估计**：高精度回归
- **RUL预测**：寿命预测
- **容量预测**：容量衰减预测

## 核心原理

**核心思想**：拟合负梯度（残差）

**步骤**：
1. 初始化：预测均值
2. 计算残差
3. 训练新模型拟合残差
4. 更新预测
5. 重复2-4

## 代码示例

```python
from sklearn.ensemble import GradientBoostingRegressor

def soh_gbm(X_train, y_train, X_test):
    """
    GBM估计SOH
    """
    gbm = GradientBoostingRegressor(
        n_estimators=100,
        learning_rate=0.1,
        max_depth=3,
        random_state=42
    )
    gbm.fit(X_train, y_train)
    soh_pred = gbm.predict(X_test)
    return soh_pred, gbm
```

## 应用场景

- **SOH估计**：高精度预测
- **RUL预测**：寿命预测
- **容量预测**：衰减趋势

✅ **100%通用**
