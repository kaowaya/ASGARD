## 算法原理

### ICA（增量容量分析）

$$
ICA = \frac{dQ}{dV} \approx \frac{\Delta Q}{\Delta V}
$$

### SOH计算

$$
SOH = \frac{Q_{current}}{Q_{nominal}} \times 100\%
$$

### 容量识别

1. 记录完整充放电曲线
2. 计算ICA曲线
3. 识别ICA峰值位置和强度
4. 根据峰值衰减估算容量损失

---

## 代码示例

```python
def calculate_soh_two_point(voltage, capacity, nominal_capacity):
    """
    两点法SOH估计
    Args:
        voltage: 充电电压序列
        capacity: 累计容量序列
        nominal_capacity: 额定容量
    Returns:
        soh: 健康状态（0-1）
    """
    # 计算ICA
    dV = np.diff(voltage)
    dQ = np.diff(capacity)
    ica = dQ / dV

    # 平滑ICA
    from scipy.signal import savgol_filter
    ica_smooth = savgol_filter(ica, window_length=21, polyorder=3)

    # 找到主峰（LFP在~3.3V）
    peak_idx = np.argmax(ica_smooth)
    peak_voltage = voltage[peak_idx]
    peak_intensity = ica_smooth[peak_idx]

    # 根据峰值强度估算SOH
    # 需要预先标定: SOH = k * peak_intensity + b
    k, b = 0.8, 0.2  # 示例系数，需实际标定

    soh = k * peak_intensity / 1000 + b  # 归一化

    return np.clip(soh, 0, 1)
```
