# F0.38 整数规划（Integer Programming）

## 算法概述

整数规划要求部分或全部变量为整数，用于组合优化问题。在电池管理系统中，IP主要用于：

- **调度问题**：充放电调度
- **组合优化**：电池选型
- **离散决策**：开关控制

## 方法

- **分支定界**（Branch and Bound）
- **割平面法**（Cutting Plane）
- **混合整数规划**（MIP）

## 代码示例

```python
from scipy.optimize import milp
import numpy as np

def charging_scheduling(battery_capacity, power_limit, time_slots):
    """
    充电调度（整数规划）

    参数:
        battery_capacity: 电池容量
        power_limit: 功率限制
        time_slots: 时间段数

    返回:
        schedule: 充电计划（离散功率档位）
    """
    # 决策变量：每个时间段的充电档位（0-10）
    n_vars = time_slots

    # 目标函数：最小化总成本（简化为充电时间）
    c = np.ones(n_vars)

    # 约束矩阵
    # 充电总量约束
    A_eq = [np.ones(n_vars)]
    b_eq = [battery_capacity]

    # 边界：整数变量
    bounds = [(0, power_limit) for _ in range(n_vars)]

    # 求解（简化版，实际需要MIP求解器）
    # 这里用LP近似，实际应使用Gurobi/CPLEX
    from scipy.optimize import linprog
    result = linprog(c, A_eq=A_eq, b_eq=b_eq, bounds=bounds)

    if result.success:
        schedule = np.round(result.x).astype(int)  # 取整
        print(f"充电计划: {schedule}")
        return schedule
    else:
        print("优化失败")
        return None

# 应用示例
if __name__ == "__main__":
    battery_capacity = 100  # 100Ah
    power_limit = 10        # 最大档位10
    time_slots = 24         # 24小时

    schedule = charging_scheduling(battery_capacity, power_limit, time_slots)
```
