## 算法原理

### 核心公式

$$
SOC(t) = SOC(t_0) - \frac{1}{C_n} \int_{t_0}^{t} \eta(I) \cdot I(\tau) d\tau
$$

其中:
- $C_n$: 额定容量（Ah）
- $\eta(I)$: 库仑效率（充电≈1，放电≈0.98-1.0）

### 库仑效率模型

$$
\eta(I, T) = \eta_0 \cdot \left[1 - \alpha \left(1 - \frac{T}{T_{ref}}\right)\right] \cdot \left[1 - \beta \left(\frac{|I|}{I_{ref}} - 1\right)\right]
$$

## 代码示例（C++嵌入式）

```cpp
class CoulombCountingSOC {
public:
    CoulombCountingSOC(float capacity_Ah, float initial_soc = 0.5)
        : Cn(capacity_Ah), soc(initial_soc), last_time(0) {}

    void update(float current_A, float temperature_C, uint32_t timestamp_ms) {
        if (last_time == 0) {
            last_time = timestamp_ms;
            return;
        }

        // 时间步长（小时）
        float dt = (timestamp_ms - last_time) / 3600000.0f;
        last_time = timestamp_ms;

        // 库仑效率
        float eta = calculate_coulombic_efficiency(current_A, temperature_C);

        // SOC更新
        float dSOC = -eta * current_A * dt / Cn;
        soc += dSOC;

        // 饱和限制
        soc = fmaxf(0.0f, fminf(1.0f, soc));
    }

    float get_soc() const { return soc; }

private:
    float Cn;              // 额定容量
    float soc;             // 当前SOC
    uint32_t last_time;    // 上次更新时间

    float calculate_coulombic_efficiency(float I, float T) {
        // 简化模型：充电η≈1，放电η随温度降低
        if (I > 0) {  // 充电
            return 1.0f;
        } else {      // 放电
            float T_ref = 25.0f;
            float alpha = 0.005f;
            return 0.99f * (1.0f - alpha * (1.0f - T / T_ref));
        }
    }
};
```
