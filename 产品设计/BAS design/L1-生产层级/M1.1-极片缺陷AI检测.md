## 算法原理

### 1. YOLOv8目标检测架构

```
输入图像（极片表面） 640×640×3
    ↓
Backbone: ResNet50特征提取
    ↓
Neck: FPN+PAN多尺度融合
    ↓
Head: 解耦检测头
    ├─ 分类分支（缺陷类型）
    └─ 回归分支（边界框）
    ↓
NMS后处理
    ↓
输出: [缺陷类型, 置信度, 坐标]
```

### 2. 缺陷分类体系

| 缺陷类型 | 特征描述 | 检测阈值 | 权重 |
|---------|---------|---------|------|
| **划痕** | 线性痕迹，长度>1mm | IoU>0.5 | 高 |
| **针孔** | 圆形穿孔，直径>0.1mm | IoU>0.3 | 高 |
| **异物** | 非规则颗粒，面积>0.5mm² | IoU>0.4 | 中 |
| **涂层不均** | 厚度变化>10μm | 纹理分析 | 中 |
| **褶皱** | 波浪状变形 | FFT特征 | 低 |

### 3. 核心公式

**CIoU Loss**（边界框回归）:

$$
L_{CIoU} = 1 - IoU + \frac{\rho^2(b, b^{gt})}{c^2} + \alpha v
$$

其中:
- $IoU = \frac{|B \cap B^{gt}|}{|B \cup B^{gt}|}$（交并比）
- $\rho(b, b^{gt})$：中心点距离
- $c$：最小包围盒对角线
- $v = \frac{4}{\pi^2}(\arctan\frac{w^{gt}}{h^{gt}} - \arctan\frac{w}{h})^2$（宽高比一致性）
- $\alpha$：权重参数

**VFL Loss**（分类）:

$$
L_{VFL} = \begin{cases}
-q \log(q) & \text{if } y=1 \\
-(1-q)^\gamma q \log(1-q) & \text{if } y=0
\end{cases}
$$

其中$q$为预测置信度，$y$为真实标签，$\gamma$为聚焦参数（默认2.0）。

## 代码示例

### 训练脚本（Python）

```python
from ultralytics import YOLO
import torch

# 加载预训练模型
model = YOLO('yolov8n.pt')

# 自定义数据集配置
data_config = {
    'train': 'datasets/electrode/train',
    'val': 'datasets/electrode/val',
    'nc': 5,  # 缺陷类别数
    'names': ['scratch', 'pinhole', 'foreign', 'coating', 'wrinkle']
}

# 训练配置
results = model.train(
    data=data_config,
    epochs=100,
    imgsz=640,
    batch=16,
    device='0' if torch.cuda.is_available() else 'cpu',
    optimizer='Adam',
    lr0=0.001,
    lrf=0.01,
    weight_decay=0.0005,
    warmup_epochs=3,
    patience=20,
    save_period=10,
    project='electrode_defect_detection',
    name='yolov8_electrode'
)

# 评估
metrics = model.val()
print(f"mAP50: {metrics.box.map50:.4f}")
print(f"mAP50-95: {metrics.box.map:.4f}")
```

### 推理脚本（Python）

```python
def detect_defects(image_path, model_path='best.pt', conf_threshold=0.5):
    """
    极片缺陷检测推理
    Args:
        image_path: 输入图像路径
        model_path: 模型权重路径
        conf_threshold: 置信度阈值
    Returns:
        defects: 检测结果列表 [{'type': 'scratch', 'conf': 0.95, 'bbox': [...]}, ...]
    """
    from ultralytics import YOLO
    import cv2

    # 加载模型
    model = YOLO(model_path)

    # 推理
    results = model(image_path, conf=conf_threshold, iou=0.5)

    # 解析结果
    defects = []
    for r in results:
        for box in r.boxes:
            defect = {
                'type': model.names[int(box.cls)],
                'conf': float(box.conf),
                'bbox': box.xyxy.tolist()[0]
            }
            defects.append(defect)

    return defects

# 使用示例
defects = detect_defects('electrode_sample.jpg')
for d in defects:
    print(f"检测到: {d['type']}, 置信度: {d['conf']:.2f}, 坐标: {d['bbox']}")
```
