# F0.42 差分进化（Differential Evolution）

## 算法概述

差分进化是一种基于差分变异的进化算法，特别适合连续优化问题。在电池管理系统中，DE主要用于：

- **参数辨识**：电池模型参数
- **连续优化**：复杂目标函数
- **全局优化**：多峰优化

## 算法步骤

```
1. 初始化种群：{x₁, x₂, ..., x_N}
2. 对每个个体x_i：
    a. 变异：v_i = x_r1 + F·(x_r2 - x_r3)
       （r1, r2, r3为随机个体，F为缩放因子）
    b. 交叉：u_i = Crossover(v_i, x_i)
    c. 选择：如果f(u_i) < f(x_i)，则x_i = u_i
3. 重复2直到收敛
```

## 代码示例

```python
import numpy as np

def differential_evolution(objective_func, bounds, n_pop=50, F=0.8, CR=0.9, max_iter=100):
    """
    差分进化优化

    参数：
        objective_func: 目标函数（最小化）
        bounds: 参数边界 [(min_1, max_1), ..., (min_n, max_n)]
        n_pop: 种群大小
        F: 缩放因子（0-2）
        CR: 交叉概率（0-1）
        max_iter: 最大迭代次数
    """
    n_vars = len(bounds)

    # 1. 初始化种群
    population = np.random.uniform(
        low=[b[0] for b in bounds],
        high=[b[1] for b in bounds],
        size=(n_pop, n_vars)
    )

    # 评估初始适应度
    fitness = np.array([objective_func(ind) for ind in population])

    # 2. 进化循环
    for iter in range(max_iter):
        for i in range(n_pop):
            # 变异：选择3个随机个体
            idxs = np.random.choice(n_pop, 3, replace=False)
            while i in idxs:
                idxs = np.random.choice(n_pop, 3, replace=False)
            x_r1, x_r2, x_r3 = population[idxs]

            # 差分变异
            v = x_r1 + F * (x_r2 - x_r3)

            # 边界处理
            v = np.clip(v, [b[0] for b in bounds], [b[1] for b in bounds])

            # 交叉
            j_rand = np.random.randint(0, n_vars)
            u = np.where(np.random.rand(n_vars) < CR, v, population[i])
            u[j_rand] = v[j_rand]  # 确保至少一个基因来自v

            # 选择
            fitness_u = objective_func(u)
            if fitness_u < fitness[i]:
                population[i] = u
                fitness[i] = fitness_u

        # 打印进度
        if iter % 10 == 0:
            best_fitness = fitness.min()
            print(f"Iteration {iter}, Best Fitness: {best_fitness:.6f}")

    # 返回最优解
    best_idx = np.argmin(fitness)
    return population[best_idx], fitness[best_idx]
```
