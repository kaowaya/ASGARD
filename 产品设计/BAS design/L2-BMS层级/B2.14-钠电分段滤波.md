## 算法原理

### 硬碳平台区特性

- 电压2.5-3.0V区间，SOC变化20%但电压几乎不变
- 传统的OCV-SOC映射失效

### 分段策略

```python
def sodium_ion_soc_filter(voltage, current, temp):
    """
    钠离子电池SOC滤波（分段处理）
    """
    # 识别平台区
    if 2.5 <= voltage <= 3.0:
        # 平台区：主要依赖安时积分
        soc_cc = coulomb_counting(current)
        return soc_cc
    else:
        # 斜坡区：可以用EKF融合OCV
        soc_ekf = ekf_estimate(voltage, current, temp)
        return soc_ekf
```
