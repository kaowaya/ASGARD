# F0.9 高斯过程回归（GPR）

## 算法概述

高斯过程回归是一种非参数贝叶斯方法，能够提供预测的不确定性估计。在电池管理系统中，GPR特别适合小样本SOH估计和贝叶斯优化。

## 核心原理

**核心思想**：假设数据来自高斯过程

**高斯过程定义**：
```
f(x) ~ GP(m(x), k(x,x'))

其中：
- m(x): 均值函数（通常为0）
- k(x,x'): 核函数（例如：RBF核）
```

**预测**：
```
f_*|X,y,x_* ~ N(μ_*, Σ_*)

μ_* = k_*^T (K + σ²I)^{-1} y
Σ_* = k(x_*, x_*) - k_*^T (K + σ²I)^{-1} k_*
```

**优势**：
- 提供不确定性估计
- 适合小样本
- 非参数模型（灵活）

## 代码示例

```python
from sklearn.gaussian_process import GaussianProcessRegressor
from sklearn.gaussian_process.kernels import RBF, ConstantKernel as C
import numpy as np

def soh_gpr(X_train, y_train, X_test):
    """
    基于GPR的SOH估计（含不确定性）
    """
    # 核函数
    kernel = C(1.0) * RBF(length_scale=1.0)

    # GPR模型
    gpr = GaussianProcessRegressor(
        kernel=kernel,
        alpha=0.1,        # 噪声水平
        normalize_y=True,
        n_restarts_optimizer=10
    )

    # 训练
    gpr.fit(X_train, y_train)

    # 预测（含不确定性）
    soh_pred, sigma = gpr.predict(X_test, return_std=True)

    # 95%置信区间
    ci_upper = soh_pred + 1.96 * sigma
    ci_lower = soh_pred - 1.96 * sigma

    return soh_pred, sigma, (ci_lower, ci_upper)
```

## 应用场景

- **小样本SOH估计**：数据稀缺时的高精度估计
- **贝叶斯优化**：参数辨识优化
- **不确定性量化**：风险评估
